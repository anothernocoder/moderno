**Promo** is an ecommerce [block](/introduction/blocks/): a sitewide announcement
bar for offers or shipping thresholds (e.g. "Free shipping over $50", "20% off
today only"), with a message, an optional CTA link, and an optional close button.
Plain layout composed with `--md-*` tokens — no primitives, no bespoke interaction
logic. You copy it into your repo with the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `message` | `string` | Sample free-shipping + discount copy | The promotional message shown in the bar |
| `ctaLabel` | `string` | `'Ver ofertas'` | Label for the optional CTA link (omit to hide it) |
| `ctaHref` | `string` | `'#'` | Href for the CTA link |
| `onCtaClick` | `(event: MouseEvent) => void` | — | Called when the CTA link is clicked |
| `dismissible` | `boolean` | `true` | Whether the close button is shown |
| `onDismiss` | `() => void` | — | Called when the bar is closed |
| `closeLabel` | `string` | `'Cerrar'` | Accessible label for the close button |

## Usage

```tsx
import { Promo } from './blocks/Promo'

<Promo
  message="Free shipping on orders over $50 — today only."
  ctaLabel="Shop now"
  ctaHref="/sale"
  onDismiss={() => trackPromoDismissed()}
/>
```

The props API is identical across all four frameworks. In Vue, `onCtaClick` and
`onDismiss` are the `@cta-click` and `@dismiss` events; in React, Svelte, and Solid
they're callback props. Dismissal is local show/hide state — no new primitive is
introduced.
