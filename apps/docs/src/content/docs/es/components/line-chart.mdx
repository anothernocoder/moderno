**Line chart** dibuja datos `{x, y}` como un path SVG, opcionalmente con un punto por
dato. Es la primitiva trazadora del chart core (ADR-0004): la geometría — escalas y el
path de la línea — se calcula una sola vez en el **chart-core** framework-neutral
(`@moderno/chart-core`, construido sobre `d3-scale` + `d3-shape`), y cada paquete de
framework solo renderiza esa geometría como SVG. El color del trazo y de los puntos
viene de tokens `--md-*`, así el theming (claro/oscuro) se hereda de CSS sin plomería
de color en JS. Primitive **closed-prop** (ADR-0003) — se pasan `data` y props de
tamaño, sin anatomía composicional.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { LineChart } from '@moderno/react'

<LineChart
  data={[
    { x: 0, y: 12 },
    { x: 1, y: 28 },
    { x: 2, y: 18 },
  ]}
  width={480}
  height={220}
  showPoints
  label="Ingresos mensuales"
/>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `data` | `{ x: number; y: number }[]` | — | Los puntos a graficar, en orden de x. |
| `width` | `number` | `480` | Ancho intrínseco del SVG en px. El gráfico se escala al 100% de su contenedor vía CSS; `width`/`height` fijan la relación de aspecto (`viewBox`). |
| `height` | `number` | `240` | Alto intrínseco del SVG en px. |
| `showPoints` | `boolean` | `false` | Renderiza un punto en cada dato. |
| `label` | `string` | — | Etiqueta accesible leída por lectores de pantalla en lugar del gráfico visual. |
