**Page/Card/Section headers** es un [block](/es/introduction/blocks/) de aplicación: una
familia de tres patrones de encabezado relacionados para vistas de aplicación — un
**encabezado de página** (breadcrumbs, título, descripción, acciones y una fila de
estado/meta), un **encabezado de tarjeta** (título, subtítulo y acciones para una
tarjeta/panel), y una **sección** con una barra superior de encabezado (título,
descripción, acciones) y una barra inferior (resumen y acciones). Compone los primitives
`Button`, `Badge`, `Card` y `Divider` y los tokens `--md-*`.

Tras copiarlo, instala el primitive `@moderno/<framework>` e
[importa los estilos](/es/introduction/installation/). A partir de ahí el código es tuyo —
separa el encabezado de página, el de tarjeta o las barras de sección en sus propios
componentes si necesitas reutilizarlos de forma independiente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `breadcrumbs` | `{ label: string; href?: string }[]` | 3 crumbs de muestra | Ruta de breadcrumbs sobre el título de página |
| `pageTitle` | `string` | `'Rediseño Q3'` | Título del encabezado de página |
| `pageDescription` | `string` | texto de muestra | Descripción del encabezado de página, bajo el título |
| `pageStatus` | `string` | `'Activo'` | Etiqueta de estado mostrada como `Badge` con punto en la fila de meta |
| `pageMeta` | `string` | texto de muestra | Texto de meta junto al badge de estado |
| `primaryLabel` | `string` | `'Nueva tarea'` | Etiqueta del botón de acción primaria del encabezado de página (oculto si está vacío) |
| `onPrimaryAction` | `() => void` | — | Se llama al hacer clic en el botón de acción primaria |
| `secondaryLabel` | `string` | `'Compartir'` | Etiqueta del botón de acción secundaria del encabezado de página (oculto si está vacío) |
| `onSecondaryAction` | `() => void` | — | Se llama al hacer clic en el botón de acción secundaria |
| `cardTitle` | `string` | `'Miembros del equipo'` | Título del encabezado de tarjeta |
| `cardSubtitle` | `string` | texto de muestra | Subtítulo del encabezado de tarjeta |
| `cardBody` | `string` | texto de muestra | Texto del cuerpo de la tarjeta, bajo un `Divider` (oculto si está vacío) |
| `cardActionLabel` | `string` | `'Invitar'` | Etiqueta del botón de acción del encabezado de tarjeta (oculto si está vacío) |
| `onCardAction` | `() => void` | — | Se llama al hacer clic en la acción del encabezado de tarjeta |
| `sectionTitle` | `string` | `'Actividad reciente'` | Título de la barra superior de sección |
| `sectionDescription` | `string` | texto de muestra | Descripción de la barra superior de sección |
| `sectionActionLabel` | `string` | `'Ver todo'` | Etiqueta de la acción de la barra superior de sección (oculto si está vacío) |
| `onSectionAction` | `() => void` | — | Se llama al hacer clic en la acción de la barra superior de sección |
| `sectionBody` | `string` | texto de muestra | Contenido del cuerpo de la sección entre ambas barras |
| `sectionSummary` | `string` | texto de muestra | Texto de resumen de la barra inferior de sección |
| `sectionBarActionLabel` | `string` | `'Exportar'` | Etiqueta de la acción secundaria de la barra inferior (oculto si está vacío) |
| `onSectionBarAction` | `() => void` | — | Se llama al hacer clic en la acción secundaria de la barra inferior |
| `sectionBarPrimaryLabel` | `string` | `'Marcar completadas'` | Etiqueta de la acción primaria de la barra inferior (oculto si está vacío) |
| `onSectionBarPrimaryAction` | `() => void` | — | Se llama al hacer clic en la acción primaria de la barra inferior |

## Uso

```tsx
import { PageCardSectionHeaders } from './blocks/PageCardSectionHeaders'

<PageCardSectionHeaders
  pageTitle="Pedido #A-102"
  pageDescription="Sigue el cumplimiento y gestiona este pedido."
  primaryLabel="Reembolsar"
  onPrimaryAction={() => refundOrder()}
  cardTitle="Cliente"
  cardActionLabel="Editar"
  onCardAction={() => editCustomer()}
  sectionTitle="Línea de tiempo"
  sectionSummary="4 eventos"
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onPrimaryAction`/
`onSecondaryAction`/`onCardAction`/`onSectionAction`/`onSectionBarAction`/
`onSectionBarPrimaryAction` son los eventos `@primary-action`/`@secondary-action`/
`@card-action`/`@section-action`/`@section-bar-action`/`@section-bar-primary-action`; en
los demás son props de callback.
