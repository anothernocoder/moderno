**Grid lists** is an application [block](/introduction/blocks/): card/tile layouts arranged
in a responsive CSS grid, each tile showing a thumbnail/icon area, title, subtitle, and an
optional status badge with an action. It composes the `Card`, `Avatar`, `Badge`, and `Button`
primitives and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `GridListItem[]` | 6 sample items | The tiles to render |
| `onItemAction` | `(item: GridListItem) => void` | — | Called when a tile's "View details" button is clicked |

`GridListItem` shape:

| Field | Type | Description |
|-------|------|--------------|
| `id` | `string` | Unique tile id |
| `title` | `string` | Tile title |
| `subtitle` | `string` | Secondary meta line (omitted if not set) |
| `image` | `string` | Thumbnail image URL, rendered by `Avatar` |
| `initials` | `string` | Fallback initials shown by `Avatar` when there is no `image` |
| `status` | `string` | Status label shown in the `Badge` (omitted if not set) |
| `statusVariant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | `Badge` color variant for the status |

## Usage

```tsx
import { GridLists } from './blocks/GridLists'

<GridLists
  items={[
    { id: '1', title: 'Checkout redesign', subtitle: 'Product team', initials: 'CR', status: 'In progress', statusVariant: 'info' },
    { id: '2', title: 'Vue 3 migration', subtitle: 'Platform team', initials: 'VM', status: 'Done', statusVariant: 'success' },
  ]}
  onItemAction={(item) => openProject(item)}
/>
```

The props API is identical across all four frameworks. In Vue, `onItemAction` is the
`@item-action` event; in the others it's a callback prop.
