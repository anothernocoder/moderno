**Combobox** is a searchable single-select: an input that filters the option list as you
type (case-insensitive contains on the label), with a trigger button to open the full list.
A **closed-prop** primitive (ADR-0003) built on Zag.js's `combobox` machine — selection,
keyboard navigation, and ARIA `combobox`/`listbox`/`option` come from Zag; Moderno owns
the markup and the `md-combobox-*` class contract.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Combobox } from '@moderno/react'

const fruits = [
  { label: 'Apple', value: 'apple' },
  { label: 'Banana', value: 'banana' },
]

<Combobox
  label="Fruit"
  placeholder="Search a fruit…"
  items={fruits}
  onValueChange={(value) => console.log(value)}
/>
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible label above the control. |
| `items` | `{ label: string; value: string; disabled?: boolean }[]` | — | Options; typing filters them by label. |
| `placeholder` | `string` | — | Input placeholder. |
| `defaultValue` | `string[]` | — | Initially selected keys (uncontrolled). |
| `value` / `v-model` / `bind:value` | `string[]` | — | Selected keys (controlled). |
| `disabled` | `boolean` | `false` | Disables the control. |
| `onValueChange` | `(value: string[]) => void` | — | Callback when the selection changes. |
