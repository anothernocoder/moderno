**Description List** es un [block](/es/introduction/blocks/) de aplicación: una lista de
detalle clave/valor para datos de registro estructurados (una vista de detalle de pedido o
perfil). Cada fila muestra una etiqueta y un valor — apilados en layouts angostos, lado a
lado en los más anchos. Compone el primitive `Card` y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `title` | `string` | `'Detalle del pedido'` | Encabezado mostrado en el título del `Card` |
| `items` | `DescriptionItem[]` | 8 campos de muestra | Pares etiqueta/valor a renderizar |

### `DescriptionItem`

| Campo | Tipo | Descripción |
|-------|------|--------------|
| `label` | `string` | Nombre del campo |
| `value` | `string` | Valor ya formateado |

## Uso

```tsx
import { DescriptionList } from './blocks/DescriptionList'

<DescriptionList
  title="Detalle del pedido"
  items={[
    { label: 'Número de pedido', value: 'A-1104' },
    { label: 'Cliente', value: 'Ada Corp' },
    { label: 'Estado', value: 'Despachado' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.
