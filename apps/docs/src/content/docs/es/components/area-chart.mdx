**Area chart** dibuja datos `{x, y}` como un área SVG rellena con una línea trazada
sobre su borde superior, opcionalmente con un punto por dato. Construye sobre la misma
geometría de **chart-core** que [Line chart](/es/components/line-chart/) (ADR-0004) —
`@moderno/chart-core`, construido sobre `d3-scale` + `d3-shape` — calculada una sola
vez y renderizada como SVG por cada paquete de framework. El relleno cierra hacia el
fondo del gráfico (no hacia el y=0 de los datos), así que se lee correctamente incluso
cuando todos los valores son positivos. El color de relleno y de trazo viene de tokens
`--md-*`. Primitive **closed-prop** (ADR-0003) — se pasan `data` y props de tamaño, sin
anatomía composicional.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { AreaChart } from '@moderno/react'

<AreaChart
  data={[
    { x: 0, y: 12 },
    { x: 1, y: 28 },
    { x: 2, y: 18 },
  ]}
  width={480}
  height={220}
  showPoints
  label="Ingresos mensuales"
/>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `data` | `{ x: number; y: number }[]` | — | Los puntos a graficar, en orden de x. |
| `width` | `number` | `480` | Ancho intrínseco del SVG en px. El gráfico se escala al 100% de su contenedor vía CSS; `width`/`height` fijan la relación de aspecto (`viewBox`). |
| `height` | `number` | `240` | Alto intrínseco del SVG en px. |
| `showPoints` | `boolean` | `false` | Renderiza un punto en cada dato. |
| `label` | `string` | — | Etiqueta accesible leída por lectores de pantalla en lugar del gráfico visual. |
