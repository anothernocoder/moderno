**Input** es un campo de texto de una línea con etiqueta, hint y error. Primitive
**closed-prop**, pura CSS. La etiqueta es siempre visible; el error reemplaza al hint y marca
el campo como inválido (`aria-invalid`).

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Input } from '@moderno/react'

<Input
  label="Correo de facturación"
  hint="Visible en tus recibos."
  placeholder="ventas@empresa.com"
/>
<Input label="Correo" error="Ingresa un correo válido." />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Input } from '@moderno/vue'
import { ref } from 'vue'
const correo = ref('')
</script>

<template>
  <Input v-model="correo" label="Correo de facturación" hint="Visible en tus recibos." />
</template>
```
</TabItem>

</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Etiqueta visible asociada al campo. |
| `hint` | `string` | — | Texto de ayuda bajo el campo. |
| `error` | `string` | — | Mensaje de error; reemplaza al hint y marca `aria-invalid`. |

Acepta además los atributos nativos de `<input>` (`type`, `placeholder`, `disabled`…). El
valor se enlaza con `v-model` (Vue) o `bind:value` (Svelte).
