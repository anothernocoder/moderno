## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Badge } from '@moderno/react'

<Badge variant="success" dot>Active</Badge>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Badge } from '@moderno/vue'
</script>

<template>
  <Badge variant="success" dot>Active</Badge>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Badge } from '@moderno/svelte'
</script>

<Badge variant="success" dot>Active</Badge>
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Badge } from '@moderno/solid'

<Badge variant="success" dot>Active</Badge>
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'neutral'` | Color/intent. |
| `dot` | `boolean` | `false` | Shows a status dot at the start. |
