**Feeds** es un [block](/es/introduction/blocks/) de aplicación: un timeline vertical de
actividad. Cada entrada muestra un avatar conectado con el siguiente por una línea de
riel, una oración de actor + acción, un tag de estado opcional y un timestamp. Compone
los primitives `Avatar` y `Badge` y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `FeedItem[]` | 4 entradas de muestra | Entradas de actividad a renderizar |

### `FeedItem`

| Campo | Tipo | Default | Descripción |
|-------|------|---------|-------------|
| `id` | `string` | — | Clave única |
| `actor` | `string` | — | Quién realizó la acción |
| `initials` | `string` | — | Iniciales de respaldo mostradas en el `Avatar` |
| `action` | `string` | — | Oración de la acción, renderizada después del nombre del actor |
| `timestamp` | `string` | — | Tiempo relativo/absoluto ya formateado |
| `tag` | `string` | — | Etiqueta de estado opcional (oculta si se omite) |
| `tagVariant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | — | Color del `Badge` para `tag` |

## Uso

```tsx
import { Feeds } from './blocks/Feeds'

<Feeds
  items={[
    { id: '1', actor: 'Ada Lovelace', initials: 'AL', action: 'fusionó un pull request', tag: 'Fusionado', tagVariant: 'success', timestamp: 'Hace 5 min' },
    { id: '2', actor: 'Grace Hopper', initials: 'GH', action: 'comentó en un issue', timestamp: 'Hace 32 min' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.
