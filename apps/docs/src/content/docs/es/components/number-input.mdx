**NumberInput** es un campo numérico con steppers `−`/`+`. Primitive **closed-prop**
(ADR-0003) sobre la máquina `number-input` de Zag.js — el paso con teclado (flechas), el
clamping min/max, el spin al mantener presionado y el ARIA `spinbutton` vienen de Zag;
Moderno es dueño del markup y del class contract `md-number-input-*`. El valor se expone
como string para preservar el formato; `onValueChange` también entrega `valueAsNumber`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Label visible sobre el control. |
| `defaultValue` | `number \| string` | — | Valor inicial (uncontrolled). |
| `value` / `v-model` / `bind:value` | `number \| string` | — | Valor controlado (internamente string). |
| `min` / `max` | `number` | — | Límites de clamping. |
| `step` | `number` | `1` | Paso de incremento/decremento. |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `name` | `string` | — | Nombre del campo (para formularios). |
| `placeholder` | `string` | — | Placeholder del input. |
| `onValueChange` | `(value: string, valueAsNumber: number) => void` | — | Callback cuando cambia el valor. |
