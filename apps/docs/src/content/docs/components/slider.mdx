**Slider** adjusts a numeric value within a range by dragging the thumb or using the keyboard. A **closed-prop** primitive (ADR-0003) built on Zag.js's `slider` machine (single thumb).

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Slider } from '@moderno/react'

<Slider label="Volume" defaultValue={60} onValueChange={(v) => console.log(v)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Slider } from '@moderno/vue'
import { ref } from 'vue'
const volume = ref(60)
</script>

<template>
  <Slider label="Volume" v-model="volume" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Slider } from '@moderno/svelte'
  let volume = $state(60)
</script>

<Slider label="Volume" bind:value={volume} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Slider } from '@moderno/solid'

<Slider label="Volume" defaultValue={60} onValueChange={(v) => console.log(v)} />
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible label. |
| `defaultValue` | `number` | `50` | Initial uncontrolled value. |
| `value` / `v-model` / `bind:value` | `number` | — | Controlled value. |
| `onValueChange` | `(value: number) => void` | — | Callback when the value changes. |
| `min` | `number` | `0` | Minimum value. |
| `max` | `number` | `100` | Maximum value. |
| `step` | `number` | `1` | Increment. |
| `disabled` | `boolean` | `false` | Disables the control. |
| `showValue` | `boolean` | `true` | Shows the value next to the label. |
| `name` | `string` | — | For form submission. |
