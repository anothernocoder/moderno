export const acciones = [
  { label: 'Editar', value: 'edit' },
  { label: 'Duplicar', value: 'duplicate' },
  { label: 'Archivar', value: 'archive' },
  { label: 'Eliminar', value: 'delete' },
]

**Menu** es un desplegable de acciones: un botón trigger que abre una lista de items, cada
uno dispara `onSelect` con su `value`. Primitive **closed-prop** (ADR-0003) sobre la máquina
`menu` de Zag.js — abrir/cerrar, teclado, typeahead, retorno de foco y el ARIA
`menu`/`menuitem` vienen de Zag; Moderno es dueño del markup y del class contract
`md-menu-*`. Para elegir un valor de formulario usa `Select` o `Combobox`.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Menu } from '@moderno/react'

const acciones = [
  { label: 'Editar', value: 'edit' },
  { label: 'Eliminar', value: 'delete' },
]

<Menu triggerLabel="Acciones" items={acciones} onSelect={(value) => console.log(value)} />
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `triggerLabel` | `string` | — | Label del botón trigger. |
| `items` | `{ label: string; value: string; disabled?: boolean }[]` | — | Las acciones que ofrece el menú. |
| `onSelect` / `@select` | `(value: string) => void` | — | Se llama con el `value` del item al seleccionarlo. |
