**Combobox** es una selección única con búsqueda: un input que filtra la lista de opciones
mientras escribes (contiene, sin distinguir mayúsculas, sobre el label), con un botón para
abrir la lista completa. Primitive **closed-prop** (ADR-0003) sobre la máquina `combobox`
de Zag.js — la selección, el teclado y el ARIA `combobox`/`listbox`/`option` vienen de Zag;
Moderno es dueño del markup y del class contract `md-combobox-*`.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Combobox } from '@moderno/react'

const frutas = [
  { label: 'Manzana', value: 'apple' },
  { label: 'Banano', value: 'banana' },
]

<Combobox
  label="Fruta"
  placeholder="Busca una fruta…"
  items={frutas}
```

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Label visible sobre el control. |
| `items` | `{ label: string; value: string; disabled?: boolean }[]` | — | Opciones; escribir las filtra por label. |
| `placeholder` | `string` | — | Placeholder del input. |
| `defaultValue` | `string[]` | — | Claves seleccionadas al inicio (uncontrolled). |
| `value` / `v-model` / `bind:value` | `string[]` | — | Claves seleccionadas (controlled). |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `onValueChange` | `(value: string[]) => void` | — | Callback cuando cambia la selección. |
