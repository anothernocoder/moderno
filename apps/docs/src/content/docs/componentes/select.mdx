**Select** elige una opción (o varias) de una lista desplegable. Primitive **compositional**
(ADR-0003): `Select.Root`, `Select.Label`, `Select.Trigger`, `Select.Content`, `Select.Item`.
El comportamiento (teclado, typeahead, foco, ARIA `listbox`/`option`, posicionamiento popper)
viene de la máquina `select` de Zag. `Root` recibe `items` y construye la colección por dentro.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Select } from '@moderno/react'

const frutas = [
  { label: 'Manzana', value: 'apple' },
  { label: 'Banano', value: 'banana' },
]

<Select.Root items={frutas} onValueChange={(v) => console.log(v)}>
  <Select.Label>Fruta</Select.Label>
  <Select.Trigger placeholder="Elige una fruta" />
  <Select.Content>
    {frutas.map((f) => <Select.Item key={f.value} item={f} />)}
  </Select.Content>
</Select.Root>
```
</TabItem>

## Props (`Select.Root`)

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `{ label: string; value: string; disabled?: boolean }[]` | — | Opciones disponibles. |
| `defaultValue` | `string[]` | — | Claves seleccionadas inicialmente (no controlado). |
| `value` / `v-model` / `bind:value` | `string[]` | — | Claves seleccionadas (controlado). |
| `multiple` | `boolean` | `false` | Permite selección múltiple. |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `onValueChange` | `(value: string[]) => void` | — | Callback al cambiar la selección. |

`Select.Trigger` acepta `placeholder`. `Select.Item` recibe `item` (un objeto de `items`).
