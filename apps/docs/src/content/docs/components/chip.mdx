import { Chip as OChip } from '@moderno/solid'

**Chip** is a compact, optionally removable token. It's a **closed-prop** primitive, pure CSS. The remove event follows the native idiom of each framework: `onRemove` prop (React/Solid), `emit('remove')` (Vue), callback prop (Svelte).

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Chip } from '@moderno/react'

<Chip variant="solid">Active</Chip>
<Chip removable onRemove={() => remove()}>Filter</Chip>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Chip } from '@moderno/vue'
</script>

<template>
  <Chip variant="solid">Active</Chip>
  <Chip removable @remove="remove">Filter</Chip>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Chip } from '@moderno/svelte'
</script>

<Chip variant="solid">Active</Chip>
<Chip removable onRemove={remove}>Filter</Chip>
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Chip } from '@moderno/solid'

<Chip variant="solid">Active</Chip>
<Chip removable onRemove={() => remove()}>Filter</Chip>
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'neutral' \| 'solid'` | `'neutral'` | Chip style. |
| `removable` | `boolean` | `false` | Shows the remove button (×). |
| `removeLabel` | `string` | `'Remove'` | `aria-label` for the remove button. |
| `onRemove` / `@remove` | `() => void` | — | Fires when the remove button is activated. |
