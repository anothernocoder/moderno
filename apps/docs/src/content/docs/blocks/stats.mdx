**Stats** es un [block](/introduccion/blocks/) de aplicación: una fila de KPIs en grid
responsivo, cada uno una tarjeta con etiqueta, valor y delta de tendencia. Compone los
primitives `Card` y `Badge` y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `StatItem[]` | 4 KPIs de muestra | Métricas a renderizar |

### `StatItem`

| Campo | Tipo | Default | Descripción |
|-------|------|---------|-------------|
| `label` | `string` | — | Nombre del KPI |
| `value` | `string` | — | Valor ya formateado |
| `delta` | `string` | — | Cambio vs. periodo anterior (oculto si se omite) |
| `trend` | `'up' \| 'down'` | — | Color del badge: `up` → success, `down` → error |

## Uso

```tsx
import { Stats } from './blocks/Stats'

<Stats
  items={[
    { label: 'Ingresos', value: '$48.2k', delta: '+12.4%', trend: 'up' },
    { label: 'Cancelaciones', value: '38', delta: '-0.8%', trend: 'down' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.
