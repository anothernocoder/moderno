**Footer** is a marketing [block](/introduction/blocks/): a site footer with a logo/tagline,
navigation link columns, legal links, and social links. It composes the `Divider` primitive and
the `--md-*` tokens. You copy it into your repo with the CLI and edit it freely.

[import the styles](/introduction/installation/). From there, the code is yours.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `logoLabel` | `string` | `'Moderno'` | Logo/wordmark text |
| `logoHref` | `string` | `'#'` | Logo link target |
| `tagline` | `string` | `'Componentes y blocks para construir producto rápido.'` | Short text under the logo |
| `columns` | `FooterColumn[]` | Three sample columns | Navigation link groups |
| `socialLinks` | `FooterLink[]` | Twitter/GitHub/LinkedIn samples | Social links row |
| `legalLinks` | `FooterLink[]` | Privacy/Terms samples | Legal links row |
| `copyright` | `string` | `'© 2026 Moderno. Todos los derechos reservados.'` | Copyright line |

A `FooterLink` is `{ label: string; href: string }`. A `FooterColumn` is
`{ title: string; links: FooterLink[] }`.

## Usage

```tsx
import { Footer } from './blocks/Footer'

<Footer
  logoLabel="Acme"
  tagline="Ship products faster."
  columns={[
    { title: 'Product', links: [{ label: 'Features', href: '/features' }, { label: 'Pricing', href: '/pricing' }] },
    { title: 'Company', links: [{ label: 'About', href: '/about' }, { label: 'Careers', href: '/careers' }] },
  ]}
  legalLinks={[{ label: 'Privacy', href: '/privacy' }, { label: 'Terms', href: '/terms' }]}
  socialLinks={[{ label: 'Twitter', href: 'https://twitter.com/acme' }]}
  copyright="© 2026 Acme Inc. All rights reserved."
/>
```

The props API is identical across all four frameworks.

The footer renders a top row with the logo/tagline on the left and the navigation columns on the
right, a `Divider`, and a bottom row with the copyright text, legal links, and social links. Omit
`tagline`, `columns`, `legalLinks`, or `socialLinks` to show fewer sections.
