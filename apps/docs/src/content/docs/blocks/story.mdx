**Story** is an images-ads [block](/introduction/blocks/): a vertical 9:16 template
for social stories and ads, with a kicker, title, and CTA. It composes the `Button` primitive
and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `kicker` | `string` | `'New collection'` | Small text on top |
| `title` | `string` | `'Design that feels modern.'` | Headline |
| `ctaLabel` | `string` | `'Shop now'` | CTA text |
| `onCta` | `() => void` | — | CTA handler |

## Usage

```tsx
import { Story } from './blocks/Story'

<Story
  kicker="New collection"
  title="Design that feels modern."
  ctaLabel="Shop now"
  onCta={() => navigate('/shop')}
/>
```

The props API is identical across all four frameworks. In Vue, `onCta` is the `@cta`
event; in the others it's a callback prop.
