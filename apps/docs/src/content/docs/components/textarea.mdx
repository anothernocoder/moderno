**Textarea** is a multi-line text field with label, hint, and error. A **closed-prop** primitive, pure CSS. It reuses the field styles from [Input](/components/input/) plus `.md-textarea`.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Textarea } from '@moderno/react'

<Textarea label="Comment" hint="500 characters max" rows={4} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Textarea } from '@moderno/vue'
import { ref } from 'vue'
const value = ref('')
</script>

<template>
  <Textarea v-model="value" label="Comment" :rows="4" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Textarea } from '@moderno/svelte'
  let value = $state('')
</script>

<Textarea bind:value label="Comment" rows={4} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Textarea } from '@moderno/solid'

<Textarea label="Comment" hint="500 characters max" rows={4} />
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label above the field. |
| `hint` | `string` | — | Helper text below the field. |
| `error` | `string` | — | Error message; replaces the hint and marks the field invalid. |

It also accepts all native `<textarea>` attributes (`rows`, `placeholder`, `disabled`, `maxlength`…). The value is bound with `v-model` (Vue) or `bind:value` (Svelte).
