**Carousel** desliza entre un conjunto de elementos a lo largo de un eje, usando CSS
scroll-snap nativo internamente. Primitive **compositional** (ADR-0003): expone partes
que comparten una sola máquina de Zag — `Carousel.Root`, `Carousel.ItemGroup`,
`Carousel.Item`, `Carousel.Control`, `Carousel.PrevTrigger`, `Carousel.NextTrigger`,
`Carousel.IndicatorGroup`, `Carousel.Indicator`. El comportamiento (paginación, loop,
arrastre, autoplay, navegación por teclado, roles ARIA `region`/`group`) viene de la
máquina `carousel`; `Carousel.Root` requiere un `slideCount` porque Zag lo necesita de
antemano para calcular las posiciones de scroll-snap.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Carousel } from '@moderno/react'

const slides = ['Uno', 'Dos', 'Tres']

<Carousel.Root slideCount={slides.length}>
  <Carousel.ItemGroup>
    {slides.map((label, index) => (
      <Carousel.Item key={label} index={index}>{label}</Carousel.Item>
    ))}
  </Carousel.ItemGroup>
  <Carousel.Control>
    <Carousel.PrevTrigger />
    <Carousel.IndicatorGroup>
      {(index) => <Carousel.Indicator key={index} index={index} />}
    </Carousel.IndicatorGroup>
    <Carousel.NextTrigger />
  </Carousel.Control>
</Carousel.Root>
```
</TabItem>

## Props (`Carousel.Root`)

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `slideCount` | `number` | — | Requerido. El número total de slides. |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | El eje sobre el que se mueven los slides. |
| `page` / `v-model:page` / `bind:page` | `number` | — | Página actual, controlada. |
| `defaultPage` | `number` | `0` | Página inicial no controlada. |
| `onPageChange` | `(page: number) => void` | — | Se llama cuando cambia la página. Vue: evento `update:page`. |
| `loop` | `boolean` | `false` | Vuelve de la última página a la primera. |
| `slidesPerPage` | `number` | `1` | Número de slides visibles a la vez. |
| `slidesPerMove` | `number \| 'auto'` | `'auto'` | Número de slides que avanza cada next/prev. |
| `autoSize` | `boolean` | `false` | Habilita slides de ancho variable. |
| `allowMouseDrag` | `boolean` | `false` | Permite arrastrar con el mouse. |
| `autoplay` | `boolean \| { delay: number }` | `false` | Avanza páginas automáticamente; el delay por default es 4000ms. |
| `spacing` | `string` | `'0px'` | Espacio entre elementos. |
| `padding` | `string` | — | Padding extra del viewport para que los slides vecinos queden parcialmente visibles. |
| `translations` | `IntlTranslations` (de `@zag-js/carousel`) | — | Mensajes de accesibilidad localizados para triggers, items e indicadores. |

`Carousel.Item` y `Carousel.Indicator` toman `index: number`. `Carousel.Control`,
`Carousel.ItemGroup`, `Carousel.PrevTrigger`, `Carousel.NextTrigger` y
`Carousel.IndicatorGroup` no necesitan props — leen la máquina compartida desde el
contexto. El contenido de `Carousel.IndicatorGroup` es una función de render
(React/Solid), un scoped slot (Vue), o un snippet parametrizado (Svelte) que se llama
una vez por cada snap point con su `index`.
