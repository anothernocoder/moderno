**KPI Cards** is an application [block](/introduction/blocks/): primary metric cards with
value, label, and a trend indicator — the single-metric counterpart to the [Stats](/blocks/stats/)
row block. It composes the `Card` and `Badge` primitives and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `KpiCardItem[]` | 3 sample KPIs | Metrics to render |

### `KpiCardItem`

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `label` | `string` | — | KPI name |
| `value` | `string` | — | Pre-formatted value |
| `delta` | `string` | — | Change vs. previous period (hidden if omitted) |
| `trend` | `'up' \| 'down'` | — | Badge color and arrow: `up` → success `↑`, `down` → error `↓` |
| `helpText` | `string` | — | Small caption next to the delta (e.g. "vs. last month") |

## Usage

```tsx
import { KpiCards } from './blocks/KpiCards'

<KpiCards
  items={[
    { label: 'Recurring revenue', value: '$128.4k', helpText: 'vs. last month', delta: '+8.2%', trend: 'up' },
    { label: 'Churn rate', value: '2.3%', helpText: 'vs. last month', delta: '-0.4%', trend: 'down' },
  ]}
/>
```

The props API is identical across all four frameworks.
