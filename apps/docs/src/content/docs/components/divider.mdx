**Divider** separates content with `role="separator"` semantics, optionally with a centered label. It's a **closed-prop** primitive, pure CSS.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx

<Divider />
<Divider label="or" />
<Divider orientation="vertical" />
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Axis of the separator. The label only applies in horizontal. |
| `label` | `string` | — | Centered label (horizontal only). |
