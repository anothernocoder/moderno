**Line chart** plots `{x, y}` data as an SVG path, optionally with a dot per point. It
is the tracer primitive for the chart core (ADR-0004): the geometry — scales and the
line path — is computed once in the framework-neutral **chart-core**
(`@moderno/chart-core`, built on `d3-scale` + `d3-shape`), and each framework package
only renders that geometry as SVG. Stroke and point color come from `--md-*` tokens, so
theming (light/dark) is inherited from CSS with no JS color plumbing. A **closed-prop**
primitive (ADR-0003) — pass `data` and size props, no compositional anatomy.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { LineChart } from '@moderno/react'

<LineChart
  data={[
    { x: 0, y: 12 },
    { x: 1, y: 28 },
    { x: 2, y: 18 },
  ]}
  width={480}
  height={220}
  showPoints
  label="Monthly revenue"
/>
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `{ x: number; y: number }[]` | — | The points to plot, in x order. |
| `width` | `number` | `480` | Intrinsic SVG width in px. The chart scales to 100% of its container via CSS; `width`/`height` set the aspect ratio (`viewBox`). |
| `height` | `number` | `240` | Intrinsic SVG height in px. |
| `showPoints` | `boolean` | `false` | Render a dot at each data point. |
| `label` | `string` | — | Accessible label read by screen readers in place of the visual chart. |
