**Toggle** is an on/off switch for binary settings that apply instantly. A **closed-prop** primitive (ADR-0003) built on Zag.js's `switch` machine: keyboard, focus, and ARIA come from the behavior core; Moderno provides the `md-*` class contract.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Toggle } from '@moderno/react'

<Toggle label="Notifications" defaultChecked onCheckedChange={(c) => console.log(c)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Toggle } from '@moderno/vue'
import { ref } from 'vue'
const on = ref(true)
</script>

<template>
  <Toggle label="Notifications" v-model="on" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Toggle } from '@moderno/svelte'
  let on = $state(true)
</script>

<Toggle label="Notifications" bind:checked={on} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Toggle } from '@moderno/solid'

<Toggle label="Notifications" defaultChecked onCheckedChange={(c) => console.log(c)} />
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible text next to the switch. |
| `defaultChecked` | `boolean` | `false` | Initial uncontrolled state. |
| `checked` / `v-model` / `bind:checked` | `boolean` | — | Controlled state. |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback on change (in Vue/Svelte use the binding). |
| `disabled` | `boolean` | `false` | Disables the control. |
| `invalid` | `boolean` | `false` | Marks the control as invalid. |
| `name` / `value` | `string` | — | For form submission. |
