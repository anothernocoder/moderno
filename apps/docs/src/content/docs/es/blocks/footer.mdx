**Footer** es un [block](/es/introduction/blocks/) de marketing: un pie de sitio con logo/tagline,
columnas de enlaces de navegación, enlaces legales y enlaces sociales. Compone el primitive
`Divider` y los tokens `--md-*`. Lo copias a tu repo con el CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `logoLabel` | `string` | `'Moderno'` | Texto del logo/wordmark |
| `logoHref` | `string` | `'#'` | Destino del enlace del logo |
| `tagline` | `string` | `'Componentes y blocks para construir producto rápido.'` | Texto corto bajo el logo |
| `columns` | `FooterColumn[]` | Tres columnas de ejemplo | Grupos de enlaces de navegación |
| `socialLinks` | `FooterLink[]` | Ejemplos de Twitter/GitHub/LinkedIn | Fila de enlaces sociales |
| `legalLinks` | `FooterLink[]` | Ejemplos de Privacidad/Términos | Fila de enlaces legales |
| `copyright` | `string` | `'© 2026 Moderno. Todos los derechos reservados.'` | Línea de copyright |

Un `FooterLink` es `{ label: string; href: string }`. Un `FooterColumn` es
`{ title: string; links: FooterLink[] }`.

## Uso

```tsx
import { Footer } from './blocks/Footer'

<Footer
  logoLabel="Acme"
  tagline="Envía producto más rápido."
  columns={[
    { title: 'Producto', links: [{ label: 'Características', href: '/features' }, { label: 'Precios', href: '/pricing' }] },
    { title: 'Compañía', links: [{ label: 'Sobre nosotros', href: '/about' }, { label: 'Carreras', href: '/careers' }] },
  ]}
  legalLinks={[{ label: 'Privacidad', href: '/privacy' }, { label: 'Términos', href: '/terms' }]}
  socialLinks={[{ label: 'Twitter', href: 'https://twitter.com/acme' }]}
  copyright="© 2026 Acme Inc. Todos los derechos reservados."
/>
```

La API de props es idéntica en los cuatro frameworks.

El footer renderiza una fila superior con el logo/tagline a la izquierda y las columnas de
navegación a la derecha, un `Divider`, y una fila inferior con el texto de copyright, los enlaces
legales y los enlaces sociales. Omite `tagline`, `columns`, `legalLinks` o `socialLinks` para
mostrar menos secciones.
