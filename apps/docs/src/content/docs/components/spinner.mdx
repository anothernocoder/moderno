**Spinner** indicates indeterminate loading. A **closed-prop** primitive, pure CSS. It carries `role="status"` and `aria-label`; the animation respects `prefers-reduced-motion`.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx

<Spinner size="lg" label="Loading data" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size (16 / 24 / 40 px). |
| `label` | `string` | `'Loading'` | Accessible label announced by screen readers. |
