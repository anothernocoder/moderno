**Table** es un [block](/es/introduction/blocks/) de aplicaciones: una tabla de datos con
columna de orden ordenable, selección por fila (con "seleccionar todo" indeterminado), badges
de estado y un footer que muestra acciones en lote cuando hay filas seleccionadas. Compone los
primitives `Checkbox`, `Badge` y `Button`, y los tokens `--md-*`. Lo copias a tu repo con el
CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `rows` | `TableRow[]` | 3 pedidos de ejemplo | Las filas a renderizar |
| `onExport` | `(selected: TableRow[]) => void` | — | Se llama al hacer clic en "Exportar", con las filas seleccionadas |
| `onCancelSelected` | `(selected: TableRow[]) => void` | — | Se llama al hacer clic en "Cancelar pedidos", con las filas seleccionadas |
| `onRowAction` | `(row: TableRow) => void` | — | Se llama al hacer clic en el botón `⋯` de una fila |

Forma de `TableRow`:

| Campo | Tipo | Descripción |
|-------|------|--------------|
| `id` | `string` | Id único de la fila, también usado como clave de selección |
| `order` | `string` | Referencia del pedido, la columna ordenable |
| `customer` | `string` | Nombre del cliente |
| `status` | `string` | Etiqueta de estado mostrada en el `Badge` |
| `statusVariant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | Variante de color del `Badge` para el estado |
| `total` | `string` | Total del pedido formateado, alineado a la derecha |

## Uso

```tsx
import { Table } from './blocks/Table'

<Table
  rows={[
    { id: 'A-1', order: 'A-1', customer: 'Acme Co', status: 'Despachado', statusVariant: 'success', total: '$120' },
    { id: 'A-2', order: 'A-2', customer: 'Globex', status: 'Pendiente', statusVariant: 'warning', total: '$40' },
  ]}
  onExport={(rows) => exportarPedidos(rows)}
  onCancelSelected={(rows) => cancelarPedidos(rows)}
  onRowAction={(row) => abrirMenuDePedido(row)}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onExport`/`onCancelSelected`/
`onRowAction` son los eventos `@export`/`@cancel-selected`/`@row-action`; en los demás son
props callback.
