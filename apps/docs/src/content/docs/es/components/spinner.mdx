**Spinner** indica carga indeterminada. Primitive **closed-prop**, pura CSS. Lleva `role="status"` y `aria-label`; la animación respeta `prefers-reduced-motion`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Tamaño (16 / 24 / 40 px). |
| `label` | `string` | `'Cargando'` | Etiqueta accesible anunciada por lectores de pantalla. |
