**NotFound** es un [block](/es/introduction/blocks/) de marketing: una sección de error 404 con un
numeral serif enorme sobre un título, una descripción y CTA(s) de vuelta al sitio. No tiene ningún
asset de ilustración — es un tratamiento puramente tipográfico, igual que el resto de blocks de
este repo evita assets binarios. Compone únicamente los tokens `--md-*`. Lo copias a tu repo con el
CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `code` | `string` | `'404'` | El numeral serif grande |
| `title` | `string` | `'Página no encontrada'` | Encabezado debajo del numeral |
| `description` | `string` | `'La página que buscas no existe o fue movida.'` | Texto de apoyo |
| `homeLabel` | `string` | `'Volver al inicio'` | Texto del enlace principal de vuelta al sitio |
| `homeHref` | `string` | `'/'` | `href` del enlace principal |
| `secondaryLabel` | `string` | `undefined` | Texto opcional para un segundo enlace estilo outline |
| `secondaryHref` | `string` | `undefined` | `href` opcional del enlace secundario |

`homeLabel`/`homeHref` y `secondaryLabel`/`secondaryHref` se renderizan como elementos `<a>` reales
(no el primitive `Button`), ya que son navegación, no callbacks. El enlace secundario solo se
renderiza cuando se proveen `secondaryLabel` y `secondaryHref`.

## Uso

```tsx
import { NotFound } from './blocks/NotFound'

<NotFound
  code="404"
  title="Página no encontrada"
  description="La página que buscas no existe o fue movida."
  homeLabel="Volver al inicio"
  homeHref="/"
  secondaryLabel="Contactar soporte"
  secondaryHref="/contacto"
/>
```

La API de props es idéntica en los cuatro frameworks.

El block renderiza una sección centrada con padding vertical generoso: el numeral `code` en una
tipografía serif enorme, `title` como un encabezado más pequeño, `description` como texto de
cuerpo, y una fila centrada de enlaces CTA — `homeHref` con apariencia de botón primario y, cuando
se provee, `secondaryHref` con apariencia de enlace secundario/outline junto a él.
