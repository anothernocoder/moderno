**Spark chart** dibuja datos `{x, y}` como una línea SVG mínima, dimensionada para uso
inline pequeño (por defecto 120×32) y que corre de borde a borde en lugar del margen
de 8px de Line chart. Construye sobre el mismo modelo de geometría de **chart-core**
que [Line chart](/es/components/line-chart/) (ADR-0004) — `@moderno/chart-core`,
construido sobre `d3-scale` + `d3-shape` — calculada una sola vez y renderizada como
SVG por cada paquete de framework. El color del trazo viene de tokens `--md-*`.
Primitive **closed-prop** (ADR-0003) — se pasan `data` y props de tamaño, sin anatomía
composicional.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { SparkChart } from '@moderno/react'

<SparkChart
  data={[
    { x: 0, y: 12 },
    { x: 1, y: 28 },
    { x: 2, y: 18 },
  ]}
  width={120}
  height={32}
  showLastPoint
  label="Tendencia de ingresos mensuales"
/>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `data` | `{ x: number; y: number }[]` | — | Los puntos a graficar, en orden de x. |
| `width` | `number` | `120` | Ancho intrínseco del SVG en px. El gráfico se escala al 100% de su contenedor vía CSS; `width`/`height` fijan la relación de aspecto (`viewBox`). |
| `height` | `number` | `32` | Alto intrínseco del SVG en px. |
| `showLastPoint` | `boolean` | `false` | Renderiza un punto en el último dato, para resaltar el valor actual. |
| `label` | `string` | — | Etiqueta accesible leída por lectores de pantalla en lugar del gráfico visual. |
