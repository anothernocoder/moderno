**NumberInput** is a numeric field with `−`/`+` steppers. A **closed-prop** primitive
(ADR-0003) built on Zag.js's `number-input` machine — keyboard stepping (arrow keys),
min/max clamping, press-and-hold spinning, and ARIA `spinbutton` come from Zag; Moderno
owns the markup and the `md-number-input-*` class contract. The value is exposed as a
string so formatted input is preserved; `onValueChange` also passes `valueAsNumber`.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible label above the control. |
| `defaultValue` | `number \| string` | — | Initial value (uncontrolled). |
| `value` / `v-model` / `bind:value` | `number \| string` | — | Controlled value (kept as string internally). |
| `min` / `max` | `number` | — | Clamping bounds. |
| `step` | `number` | `1` | Increment/decrement step. |
| `disabled` | `boolean` | `false` | Disables the control. |
| `name` | `string` | — | Form field name (for submission). |
| `placeholder` | `string` | — | Input placeholder. |
| `onValueChange` | `(value: string, valueAsNumber: number) => void` | — | Callback when the value changes. |
