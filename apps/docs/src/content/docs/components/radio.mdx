**Radio** offers single selection within a group of mutually exclusive options. A **closed-prop** primitive (ADR-0003) built on Zag.js's `radio-group` machine: pass it an `options` array and a group `label`.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Radio } from '@moderno/react'

const options = [
  { value: 'card', label: 'Card' },
  { value: 'cash', label: 'Cash' },
]

<Radio label="Payment method" options={options} defaultValue="card" onValueChange={(v) => console.log(v)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Radio } from '@moderno/vue'
import { ref } from 'vue'
const value = ref('card')
const options = [
  { value: 'card', label: 'Card' },
  { value: 'cash', label: 'Cash' },
]
</script>

<template>
  <Radio label="Payment method" :options="options" v-model="value" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Radio } from '@moderno/svelte'
  let value = $state('card')
  const options = [
    { value: 'card', label: 'Card' },
    { value: 'cash', label: 'Cash' },
  ]
</script>

<Radio label="Payment method" {options} bind:value />
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `options` | `{ value: string; label?: string; disabled?: boolean }[]` | — | Options in the group. |
| `label` | `string` | — | Group label. |
| `defaultValue` | `string` | — | Initial uncontrolled value. |
| `value` / `v-model` / `bind:value` | `string` | — | Controlled value. |
| `onValueChange` | `(value: string \| null) => void` | — | Callback when the value changes. |
| `disabled` | `boolean` | `false` | Disables the entire group. |
| `invalid` | `boolean` | `false` | Marks the group as invalid. |
| `orientation` | `'horizontal' \| 'vertical'` | `'vertical'` | Layout of the options. |
| `name` | `string` | — | For form submission. |
