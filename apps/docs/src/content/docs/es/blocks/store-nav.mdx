**Store Nav** es un [block](/es/introduction/blocks/) de ecommerce: una barra de
navegación de encabezado de tienda con logo, enlaces de categorías, un campo de
búsqueda y un ícono de carrito con un badge de cantidad de artículos. Compone los
primitives `Badge` e `Input`, y los tokens `--md-*`. Lo copias a tu repo con el CLI
y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `logoLabel` | `string` | `'Moderno Store'` | Nombre de la tienda mostrado como enlace del logo |
| `logoHref` | `string` | `'#'` | Destino del enlace del logo |
| `categories` | `{ label: string; href: string }[]` | 4 categorías de ejemplo | Enlaces de navegación de categorías |
| `searchPlaceholder` | `string` | `'Buscar productos...'` | Texto de placeholder del campo de búsqueda |
| `defaultSearchQuery` | `string` | `''` | Valor inicial del campo de búsqueda |
| `onSearch` | `(query: string) => void` | — | Se llama al enviar el formulario de búsqueda, con la consulta actual |
| `cartCount` | `number` | `3` | Cantidad de artículos mostrada en el badge del carrito (se oculta si es `0`) |
| `onCartClick` | `() => void` | — | Se llama al hacer clic en el ícono del carrito |

## Uso

```tsx
import { StoreNav } from './blocks/StoreNav'

<StoreNav
  logoLabel="Nordic Goods"
  categories={[
    { label: 'Nuevo', href: '/nuevo' },
    { label: 'Mujer', href: '/mujer' },
    { label: 'Hombre', href: '/hombre' },
    { label: 'Ofertas', href: '/ofertas' },
  ]}
  cartCount={carrito.items.length}
  onSearch={(query) => router.push(`/buscar?q=${query}`)}
  onCartClick={() => carrito.abrir()}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onSearch` y
`onCartClick` son los eventos `@search` y `@cart-click`; en los demás son props
callback.
