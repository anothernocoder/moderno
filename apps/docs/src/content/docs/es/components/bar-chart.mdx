**Bar chart** dibuja datos `{label, value}` como una barra SVG por dato, desde una
línea base compartida en 0 — los valores negativos crecen hacia abajo desde la línea
base en lugar de recortarse. Construye sobre el mismo modelo de geometría de
**chart-core** que [Line chart](/es/components/line-chart/) (ADR-0004) —
`@moderno/chart-core`, construido sobre `d3-scale` + `d3-shape` — calculada una sola
vez y renderizada como SVG por cada paquete de framework. El relleno de las barras
viene de tokens `--md-*`. Primitive **closed-prop** (ADR-0003) — se pasan `data` y
props de tamaño, sin anatomía composicional.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { BarChart } from '@moderno/react'

<BarChart
  data={[
    { label: 'Ene', value: 12 },
    { label: 'Feb', value: 28 },
    { label: 'Mar', value: 18 },
  ]}
  width={480}
  height={220}
  label="Altas mensuales"
/>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `data` | `{ label: string; value: number }[]` | — | Las barras a graficar, en orden de presentación. |
| `width` | `number` | `480` | Ancho intrínseco del SVG en px. El gráfico se escala al 100% de su contenedor vía CSS; `width`/`height` fijan la relación de aspecto (`viewBox`). |
| `height` | `number` | `240` | Alto intrínseco del SVG en px. |
| `label` | `string` | — | Etiqueta accesible leída por lectores de pantalla en lugar del gráfico visual. |
