**Indicator** is a status dot with an optional pulse and label. It's a **closed-prop** primitive, pure CSS. Unlike the [Badge](/components/badge/) dot, it can animate to signal live activity.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'neutral' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'neutral'` | Color/intent of the dot. |
| `pulse` | `boolean` | `false` | Animates a pulsing ring (respects `prefers-reduced-motion`). |
| `children` | `ReactNode` | — | Optional label after the dot. |
