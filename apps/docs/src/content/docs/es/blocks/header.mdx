**Header** es un [block](/es/introduction/blocks/) de marketing: una barra de navegación /
encabezado de sitio con logo, enlaces de navegación y CTA(s). Compone el primitive `Button`
y los tokens `--md-*`. Lo copias a tu repo con el CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `logoLabel` | `string` | `'Moderno'` | El texto del logo |
| `logoHref` | `string` | `'#'` | Adónde enlaza el logo |
| `navItems` | `{ label: string; href: string }[]` | Tres enlaces de ejemplo | Los enlaces de navegación de la barra superior |
| `secondaryLabel` | `string` | `'Iniciar sesión'` | Texto del enlace secundario. Pásalo como string vacío para ocultarlo |
| `secondaryHref` | `string` | `'#'` | Adónde apunta el enlace secundario |
| `ctaLabel` | `string` | `'Empezar ahora'` | Texto del CTA principal. Pásalo como string vacío para ocultar el CTA |
| `ctaHref` | `string?` | — | Renderiza el CTA como enlace en vez de botón |
| `onCtaClick` | `() => void` | — | Se llama al hacer clic en el botón del CTA (se ignora si `ctaHref` está definido) |

## Uso

```tsx
import { Header } from './blocks/Header'

<Header
  logoLabel="Acme"
  navItems={[
    { label: 'Producto', href: '/producto' },
    { label: 'Precios', href: '/precios' },
    { label: 'Docs', href: '/docs' },
  ]}
  secondaryLabel="Iniciar sesión"
  secondaryHref="/iniciar-sesion"
  ctaLabel="Empezar ahora"
  ctaHref="/registro"
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onCtaClick` es el evento
`@cta`; en los demás es una prop callback.

La barra pasa a una segunda línea en viewports angostos (`flex-wrap`) en vez de colapsar en
un menú hamburguesa — agrega un primitive de disclosure/menu alrededor del nav si necesitas
uno.
