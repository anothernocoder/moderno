**Chip** es un token compacto, opcionalmente eliminable. Primitive **closed-prop**, pura CSS. El evento de borrado sigue el idioma nativo: prop `onRemove` (React/Solid), `emit('remove')` (Vue), prop callback (Svelte).

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Chip } from '@moderno/react'

<Chip variant="solid">Activo</Chip>
<Chip removable onRemove={() => quitar()}>Filtro</Chip>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `variant` | `'neutral' \| 'solid'` | `'neutral'` | Estilo del chip. |
| `removable` | `boolean` | `false` | Muestra el botón de borrado (×). |
| `onRemove` / `@remove` | `() => void` | — | Se dispara al activar el botón de borrado. |
