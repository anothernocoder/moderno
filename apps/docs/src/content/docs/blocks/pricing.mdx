**Pricing** is a marketing [block](/introduction/blocks/): a plan comparison section with a
kicker, title, subtitle, and a grid of pricing cards. It composes the `Card`, `Badge`,
`Button`, and `Divider` primitives and the `--md-*` tokens. You copy it into your repo with
the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `kicker` | `string` | `'Precios'` | Small text above the title |
| `title` | `string` | `'A plan for every stage'` | Main headline |
| `subtitle` | `string` | `'Start free…'` | Supporting text |
| `plans` | `PricingPlan[]` | 3 sample plans | The cards to render |
| `onSelect` | `(plan: PricingPlan) => void` | — | Called when a plan's CTA is clicked |

`PricingPlan` shape:

| Field | Type | Description |
|-------|------|--------------|
| `name` | `string` | Plan name (also the `Card` title) |
| `price` | `string` | Headline price, e.g. `'$29'` |
| `period` | `string?` | Billing period, e.g. `'/mo'` |
| `description` | `string?` | One-line plan summary |
| `features` | `string[]` | Bulleted feature list |
| `ctaLabel` | `string?` | Button label (default `'Choose plan'`) |
| `featured` | `boolean?` | Highlights the card with a "Popular" badge and a primary CTA |

## Usage

```tsx
import { Pricing } from './blocks/Pricing'

<Pricing
  kicker="Pricing"
  title="Pick your plan"
  subtitle="Cancel anytime."
  plans={[
    { name: 'Starter', price: '$0', features: ['1 project'] },
    { name: 'Pro', price: '$29', period: '/mo', features: ['Unlimited projects'], featured: true },
  ]}
  onSelect={(plan) => track('pricing_cta_click', { plan: plan.name })}
/>
```

The props API is identical across all four frameworks. In Vue, `onSelect` is the `@select`
event; in the others it's a callback prop.
