**Product Lists** es un [block](/es/introduction/blocks/) de ecommerce: un grid de
tiles de producto con encabezado de sección y paginación simple prev/next. Compone
los mismos primitives `Card` + `Badge` + `Button` y el mismo markup de tile que el
block [Product Card](/es/blocks/product-card/), y los tokens `--md-*`. Lo copias a tu
repo con el CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `heading` | `string` | `'Productos destacados'` | Encabezado sobre el grid (omítelo para ocultarlo) |
| `products` | `ProductListItem[]` | 9 productos de ejemplo | Productos a renderizar — cada uno con `id`, `image`, `title`, `price` y `saleText` opcional |
| `pageSize` | `number` | `6` | Productos mostrados por página |
| `prevLabel` | `string` | `'Anterior'` | Texto del botón de página anterior |
| `nextLabel` | `string` | `'Siguiente'` | Texto del botón de página siguiente |
| `onAdd` | `(product: ProductListItem, index: number) => void` | — | Se llama al hacer clic en "Agregar al carrito" de un tile |

## Uso

```tsx
import { ProductLists } from './blocks/ProductLists'

<ProductLists
  heading="Productos destacados"
  products={[
    { id: 'silla', title: 'Silla Nórdica', price: '$320' },
    { id: 'mesa', title: 'Mesa Roble', price: '$540', saleText: '-10%' },
  ]}
  pageSize={6}
  onAdd={(product) => agregarAlCarrito(product)}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onAdd` es el evento
`@add`; en los demás es una prop callback. La paginación es estado local del
componente — un par de botones prev/next y un indicador de página — que divide
`products` en páginas de `pageSize`; los controles solo se renderizan cuando hay más
de una página. Sin primitives nuevos, sin lógica de interacción a medida.
