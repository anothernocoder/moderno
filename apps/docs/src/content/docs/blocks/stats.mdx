**Stats** is an application [block](/introduction/blocks/): a row of KPIs in a responsive
grid, each one a card with a label, value, and trend delta. It composes the
`Card` and `Badge` primitives and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `StatItem[]` | 4 sample KPIs | Metrics to render |

### `StatItem`

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `label` | `string` | — | KPI name |
| `value` | `string` | — | Pre-formatted value |
| `delta` | `string` | — | Change vs. previous period (hidden if omitted) |
| `trend` | `'up' \| 'down'` | — | Badge color: `up` → success, `down` → error |

## Usage

```tsx
import { Stats } from './blocks/Stats'

<Stats
  items={[
    { label: 'Revenue', value: '$48.2k', delta: '+12.4%', trend: 'up' },
    { label: 'Cancellations', value: '38', delta: '-0.8%', trend: 'down' },
  ]}
/>
```

The props API is identical across all four frameworks.
