**Long Form Content** es un [block](/es/introduction/blocks/) de portafolio: un layout
de caso de estudio / lectura larga con kicker (categoría + tiempo de lectura), headline,
dek (subtítulo), imagen de portada y secciones de prosa — cada una con subtítulo,
párrafos y pull-quote opcionales. Está construido solo con los tokens `--md-*`, sin
primitives.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'Caso de estudio'` | Etiqueta de categoría antes del tiempo de lectura |
| `readingTime` | `string` | `'12 min'` | Tiempo de lectura mostrado después del kicker |
| `headline` | `string` | headline de ejemplo | Headline principal (`h2`) |
| `dek` | `string` | dek de ejemplo | Subtítulo debajo del headline |
| `coverImageSrc` | `string` | — | URL de la imagen de portada (placeholder si se omite) |
| `coverImageAlt` | `string` | `'Imagen de portada'` | Texto alternativo de la imagen de portada |
| `sections` | `LongFormSection[]` | 1 sección de ejemplo | Secciones de prosa del cuerpo |

### `LongFormSection`

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `subheading` | `string` (opcional) | Subtítulo de la sección (`h3`) |
| `paragraphs` | `string[]` | Texto de los párrafos, en orden |
| `pullQuote` | `string` (opcional) | Pull-quote que se muestra tras el primer párrafo |

## Uso

```tsx
import { LongFormContent } from './blocks/LongFormContent'

<LongFormContent
  kicker="Caso de estudio"
  readingTime="8 min"
  headline="Cómo rediseñamos el checkout"
  dek="De un embudo de cinco pasos a uno de dos, sin bajar la conversión."
  coverImageSrc="/images/checkout-cover.jpg"
  coverImageAlt="Captura del nuevo checkout"
  sections={[
    {
      subheading: 'El contexto',
      paragraphs: ['El checkout original perdía el 40% de los usuarios en el paso de pago.'],
      pullQuote: 'La fricción no estaba en el diseño: estaba en el orden de las preguntas.',
    },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.
