**Grid lists** es un [block](/es/introduction/blocks/) de aplicación: tarjetas/tiles
dispuestas en un grid CSS responsivo, cada una con área de miniatura/icono, título,
subtítulo y un badge de estado opcional con una acción. Compone los primitives `Card`,
`Avatar`, `Badge` y `Button`, y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `GridListItem[]` | 6 items de muestra | Los tiles a renderizar |
| `onItemAction` | `(item: GridListItem) => void` | — | Se llama al hacer clic en "Ver detalle" de un tile |

Forma de `GridListItem`:

| Campo | Tipo | Descripción |
|-------|------|--------------|
| `id` | `string` | Id único del tile |
| `title` | `string` | Título del tile |
| `subtitle` | `string` | Línea secundaria de meta (se omite si no se define) |
| `image` | `string` | URL de la imagen de miniatura, renderizada por `Avatar` |
| `initials` | `string` | Iniciales de respaldo que muestra `Avatar` cuando no hay `image` |
| `status` | `string` | Etiqueta de estado mostrada en el `Badge` (se omite si no se define) |
| `statusVariant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | Variante de color del `Badge` para el estado |

## Uso

```tsx
import { GridLists } from './blocks/GridLists'

<GridLists
  items={[
    { id: '1', title: 'Rediseño de checkout', subtitle: 'Equipo de Producto', initials: 'RC', status: 'En curso', statusVariant: 'info' },
    { id: '2', title: 'Migración a Vue 3', subtitle: 'Equipo de Plataforma', initials: 'MV', status: 'Completado', statusVariant: 'success' },
  ]}
  onItemAction={(item) => openProject(item)}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onItemAction` es el evento
`@item-action`; en los demás es un callback prop.
