**Portfolio Header** es un [block](/es/introduction/blocks/) de portafolio: un encabezado de
presentación con avatar, nombre, rol, bio, enlaces sociales y un CTA de contacto. Compone
los primitives `Avatar`, `Chip` y `Button` y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `name` | `string` | `'Ana Restrepo'` | Nombre mostrado |
| `role` | `string` | `'Diseñadora de producto'` | Rol o título |
| `bio` | `string` | `'Construyo interfaces…'` | Descripción corta |
| `avatarSrc` | `string` | — | URL del avatar (usa iniciales si se omite) |
| `initials` | `string` | `'AR'` | Fallback del avatar |
| `links` | `PortfolioLink[]` | GitHub · Dribbble · LinkedIn | Enlaces sociales |
| `onContact` | `() => void` | — | Handler del CTA de contacto |

### `PortfolioLink`

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `label` | `string` | Texto del enlace |
| `href` | `string` | URL destino |

## Uso

```tsx
import { PortfolioHeader } from './blocks/PortfolioHeader'

<PortfolioHeader
  name="Ana Restrepo"
  role="Diseñadora de producto"
  initials="AR"
  links={[{ label: 'GitHub', href: 'https://github.com/ana' }]}
  onContact={() => openMailto()}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onContact` es el evento
`@contact`; en los demás es una prop callback.
