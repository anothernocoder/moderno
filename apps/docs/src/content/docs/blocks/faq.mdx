**Faq** is a marketing [block](/introduction/blocks/): a frequently-asked-questions section with a
centered eyebrow/title/subtitle header above a Q&A list. It composes the `Accordion` primitive and
the `--md-*` tokens. You copy it into your repo with the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `eyebrow` | `string` | `'Preguntas frecuentes'` | Small label above the title |
| `title` | `string` | `'¿Tienes dudas?'` | Section title |
| `subtitle` | `string` | `'Encuentra respuestas a las preguntas más frecuentes.'` | Short intro text |
| `items` | `FaqItem[]` | Five sample questions | The Q&A list |
| `defaultValue` | `string[]` | `['faq-0']` | Accordion item ids expanded on mount |

A `FaqItem` is `{ question: string; answer: string }`. Each item is rendered as an `Accordion.Item`
with a generated `value` of `faq-${index}`.

## Usage

```tsx
import { Faq } from './blocks/Faq'

<Faq
  eyebrow="FAQ"
  title="Got questions?"
  subtitle="Answers to the questions we hear most often."
  items={[
    { question: 'Do I need a credit card to start?', answer: 'No, the free plan needs no payment method.' },
    { question: 'Can I cancel anytime?', answer: 'Yes, cancel from your account settings whenever you like.' },
  ]}
/>
```

The props API is identical across all four frameworks.

The block renders a centered header (eyebrow, title, subtitle) above an `Accordion.Root` built
from `items`, one `Accordion.Item` per question. By default the first item is expanded and the
accordion is `collapsible`, so any item can be closed again; pass `defaultValue={[]}` to start with
everything collapsed.
