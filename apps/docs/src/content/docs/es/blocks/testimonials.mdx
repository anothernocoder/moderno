**Testimonials** es un [block](/es/introduction/blocks/) de marketing: una sección de prueba
social con un encabezado centrado (kicker/título/subtítulo) sobre una grilla de citas de clientes,
cada una con nombre de autor, rol y avatar. Compone los primitives `Card` y `Avatar` y los tokens
`--md-*`. Lo copias a tu repo con el CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'Testimonios'` | Texto pequeño sobre el título |
| `title` | `string` | `'Lo que dicen nuestros clientes'` | Título de la sección |
| `subtitle` | `string` | `'Equipos de todos los tamaños construyen con Moderno.'` | Texto de introducción corto |
| `items` | `TestimonialItem[]` | Tres testimonios de ejemplo | Las tarjetas de cita |

Un `TestimonialItem` es `{ quote: string; name: string; role?: string; avatarSrc?: string; initials?: string }`.
Cada item se renderiza como un `Card` con la cita como cuerpo y el autor (`Avatar` + nombre + rol)
como footer. Cuando falta `avatarSrc`, `Avatar` recurre a `initials`.

## Uso

```tsx
import { Testimonials } from './blocks/Testimonials'

<Testimonials
  kicker="Testimonios"
  title="Equipos de todos los tamaños confían en nosotros"
  subtitle="Esto es lo que dicen nuestros clientes."
  items={[
    {
      quote: 'Moderno nos permitió lanzar el rediseño en dos semanas.',
      name: 'Jamie Chen',
      role: 'Head of Design, Nimbus',
      initials: 'JC',
    },
    {
      quote: 'Una sola API en React, Vue, Svelte y Solid: una gran ventaja para el equipo.',
      name: 'Alex Rivera',
      role: 'Frontend Lead, Kairos',
      initials: 'AR',
    },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.

El block renderiza un encabezado centrado (kicker, título, subtítulo) sobre una grilla responsiva
de `Card`s construida a partir de `items`, una tarjeta por testimonio. La grilla usa
`repeat(auto-fit, minmax(280px, 1fr))`, así que las tarjetas se reacomodan de tres columnas a una
sola a medida que el viewport se angosta.
