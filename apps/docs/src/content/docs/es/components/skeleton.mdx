import { Skeleton as OSkeleton } from '@moderno/solid'

**Skeleton** es un placeholder de carga con animación shimmer. Es decorativo (`aria-hidden`), pura CSS. Las animaciones respetan `prefers-reduced-motion`.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Skeleton } from '@moderno/react'

<Skeleton variant="circle" width={48} height={48} />
<Skeleton variant="text" width="80%" />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Skeleton } from '@moderno/vue'
</script>

<template>
  <Skeleton variant="circle" :width="48" :height="48" />
  <Skeleton variant="text" width="80%" />
</template>
```
</TabItem>
<TabItem label="Svelte">

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `variant` | `'text' \| 'rect' \| 'circle'` | `'text'` | Forma del placeholder. |
| `width` | `string \| number` | — | Ancho CSS; un número se interpreta en px. |
| `height` | `string \| number` | — | Alto CSS; un número se interpreta en px. |
