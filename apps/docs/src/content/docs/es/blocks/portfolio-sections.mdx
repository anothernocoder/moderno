**Portfolio Sections** es un [block](/es/introduction/blocks/) de portafolio: un grid de
trabajo con tarjetas de proyecto que incluyen imagen, título, categoría/fecha y una flecha
que aparece al pasar el mouse enlazando al caso de estudio. Está construido solo con los
tokens `--md-*`, sin primitives.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `projects` | `PortfolioProject[]` | 4 proyectos de ejemplo | Proyectos a mostrar en el grid |
| `onSelect` | `(project: PortfolioProject) => void` | — | Se llama al hacer click en una tarjeta |

### `PortfolioProject`

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `title` | `string` | Título del proyecto |
| `meta` | `string` | Línea de categoría / fecha |
| `image` | `string` | URL de la imagen de portada (placeholder si se omite) |
| `href` | `string` | URL destino del caso de estudio |

## Uso

```tsx
import { PortfolioSections } from './blocks/PortfolioSections'

<PortfolioSections
  projects={[
    { title: 'Sistema de diseño Moderno', meta: 'Design systems · 2025', href: '/trabajo/moderno' },
    { title: 'Checkout de Rappi Turbo', meta: 'Producto · 2024', href: '/trabajo/rappi-turbo' },
  ]}
  onSelect={(project) => track('portfolio_card_click', project)}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onSelect` es el evento
`@select`; en los demás es una prop callback.
