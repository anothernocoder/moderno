**Product Card** is an ecommerce [block](/introduction/blocks/): a product card
with an image, an optional sale badge, title, price, and an "Add to cart" CTA.
It composes the `Card`, `Badge`, and `Button` primitives and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `image` | `string` | ŌĆö | Image URL (placeholder if omitted) |
| `title` | `string` | `'Moderno Chair'` | Product name |
| `price` | `string` | `'$320'` | Pre-formatted price |
| `saleText` | `string` | ŌĆö | Sale badge text (hidden if omitted) |
| `onAdd` | `() => void` | ŌĆö | "Add to cart" handler |

## Usage

```tsx
import { ProductCard } from './blocks/ProductCard'

<ProductCard
  image="/img/silla.jpg"
  title="Moderno Chair"
  price="$256"
  saleText="-20%"
  onAdd={() => cart.add('chair')}
/>
```

The props API is identical across all four frameworks. In Vue, `onAdd` is the `@add`
event; in the others it's a callback prop.
