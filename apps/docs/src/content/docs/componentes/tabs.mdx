**Tabs** permite cambiar entre vistas hermanas dentro del mismo contexto. Es un primitive
**compositional** (ADR-0003): expone partes que comparten una sola máquina de estado de Zag
— `Tabs.Root`, `Tabs.List`, `Tabs.Trigger`, `Tabs.Content`. El comportamiento (teclado con
flechas, `Home`/`End`, foco, ARIA `role="tab"`/`tabpanel`) viene de la máquina `tabs` de Zag.

## Demo en vivo

El mismo componente, renderizado desde cada framework. Cambia de pestaña con el mouse o con
las flechas del teclado.

## Anatomía y props

| Parte | Prop | Descripción |
|-------|------|-------------|
| `Tabs.Root` | `defaultValue` | Pestaña seleccionada inicialmente (no controlada). |
| `Tabs.Root` | `value` | Pestaña seleccionada (controlada). |
| `Tabs.Root` | `onValueChange` | Callback `(value: string) => void` al cambiar de pestaña. |
| `Tabs.Trigger` | `value`, `disabled` | Identifica la pestaña; `disabled` la inhabilita. |
| `Tabs.Content` | `value` | Asocia el panel a la pestaña con el mismo `value`. |

En Vue, `value` controlado se enlaza con `v-model`; en Svelte, con `bind:value`.

## Accesibilidad

- ARIA completo (`role="tablist"`/`tab"`/`tabpanel"`, `aria-selected`, `aria-controls`)
  provisto por la máquina `tabs` de Zag.
- Navegación por teclado: flechas para moverse, `Home`/`End` para ir a los extremos.
- El panel activo es enfocable; el indicador de foco usa `--md-primary`.
