---
title: Callout
description: Bloque destacado que llama la atención sobre un mensaje o consejo. Pura CSS, closed-prop.
---

import { Tabs, TabItem } from '@astrojs/starlight/components'

import { Callout as SCallout } from '@moderno/svelte'
import { Callout as OCallout } from '@moderno/solid'

**Callout** resalta un mensaje o consejo. Primitive **closed-prop**, pura CSS.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Callout } from '@moderno/react'

<Callout variant="warning" title="Atención">
  Esta acción afecta a todos los miembros del equipo.
</Callout>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Callout } from '@moderno/vue'
</script>

<template>
  <Callout variant="warning" title="Atención">
    Esta acción afecta a todos los miembros del equipo.
  </Callout>
</template>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Color del borde/intención. |
| `title` | `string` | — | Título opcional en negrita. |
