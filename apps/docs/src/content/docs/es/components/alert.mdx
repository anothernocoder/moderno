---
title: Alert
description: Mensaje de feedback inline (info/éxito/advertencia/error). Pura CSS, closed-prop.
---

import { Tabs, TabItem } from '@astrojs/starlight/components'
import { Alert as RAlert } from '@moderno/react'
import { Alert as VAlert } from '@moderno/vue'
import { Alert as SAlert } from '@moderno/svelte'
import { Alert as OAlert } from '@moderno/solid'

**Alert** muestra feedback inline para la vista actual. Primitive **closed-prop**, pura CSS
— sin máquina de Zag, consistente con [Callout](/es/components/callout/).

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Alert } from '@moderno/react'

<Alert variant="error" title="Error">
  No se pudo guardar: el stock mínimo no puede superar el punto de reorden.
</Alert>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Intención, define el color del borde. |
| `title` | `string` | — | Título opcional en negrita. |
