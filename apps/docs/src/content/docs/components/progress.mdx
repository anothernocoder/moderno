**Progress** shows the completion status of a task — linear bar or circular ring, determinate
(a known percentage) or indeterminate (unknown duration). A **closed-prop** primitive
(ADR-0003) built on Zag.js's `progress` machine.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Progress } from '@moderno/react'

<Progress label="Uploading" value={40} />
<Progress label="Connecting" value={null} variant="circular" />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Progress } from '@moderno/vue'
import { ref } from 'vue'
const uploaded = ref(40)
</script>

<template>
  <Progress label="Uploading" v-model="uploaded" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Progress } from '@moderno/svelte'
  let uploaded = $state(40)
</script>

<Progress label="Uploading" bind:value={uploaded} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Progress } from '@moderno/solid'

<Progress label="Uploading" value={40} />
<Progress label="Connecting" value={null} variant="circular" />
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible label. |
| `defaultValue` | `number \| null` | `50` | Initial uncontrolled value. `null` starts indeterminate. |
| `value` / `v-model` / `bind:value` | `number \| null` | — | Controlled value. `null` renders an indeterminate bar/ring. |
| `onValueChange` | `(value: number \| null) => void` | — | Callback when the value changes. |
| `min` | `number` | `0` | Minimum value. |
| `max` | `number` | `100` | Maximum value. |
| `variant` | `'linear' \| 'circular'` | `'linear'` | Bar or ring rendering. |
| `showValue` | `boolean` | `true` | Shows the formatted value text next to the label. |
