**Input** is a single-line text field with label, hint, and error. It's a **closed-prop**
primitive, pure CSS. The label is always visible; the error replaces the hint and marks
the field as invalid (`aria-invalid`).

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Input } from '@moderno/react'

<Input
  label="Billing email"
  hint="Visible on your receipts."
  placeholder="sales@company.com"
/>
<Input label="Email" error="Enter a valid email." />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Input } from '@moderno/vue'
import { ref } from 'vue'
const email = ref('')
</script>

<template>
  <Input v-model="email" label="Billing email" hint="Visible on your receipts." />
</template>
```
</TabItem>
<TabItem label="Svelte">

</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Visible label associated with the field. |
| `hint` | `string` | — | Help text below the field. |
| `error` | `string` | — | Error message; replaces the hint and sets `aria-invalid`. |

It also accepts the native `<input>` attributes (`type`, `placeholder`, `disabled`…). The
value is bound with `v-model` (Vue) or `bind:value` (Svelte).
