**Bar list chart** ranks `{label, value}` data as horizontal rows, each a track the
full width of the chart with a fill proportional to its share of the largest value —
label and value are rendered as SVG text alongside the fill. It builds on the same
**chart-core** geometry model as [Line chart](/components/line-chart/) (ADR-0004) —
`@moderno/chart-core`, built on `d3-scale` + `d3-shape` — computed once and rendered as
SVG by each framework package. Track, fill, and text color come from `--md-*` tokens. A
**closed-prop** primitive (ADR-0003) — pass `data` and size props, no compositional
anatomy. `data` is not re-sorted, so pass it pre-ranked for a conventional bar list.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { BarListChart } from '@moderno/react'

<BarListChart
  data={[
    { label: 'Direct', value: 456 },
    { label: 'Organic search', value: 351 },
    { label: 'Referral', value: 271 },
  ]}
  width={480}
  height={220}
  label="Traffic by source"
/>
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `{ label: string; value: number }[]` | — | The rows to plot, in display order (not re-sorted). |
| `width` | `number` | `480` | Intrinsic SVG width in px. The chart scales to 100% of its container via CSS; `width`/`height` set the aspect ratio (`viewBox`). |
| `height` | `number` | `240` | Intrinsic SVG height in px. |
| `label` | `string` | — | Accessible label read by screen readers in place of the visual chart. |
