**Bar list chart** clasifica datos `{label, value}` como filas horizontales, cada una
una pista del ancho completo del gráfico con un relleno proporcional a su parte del
valor más grande — la etiqueta y el valor se renderizan como texto SVG junto al
relleno. Construye sobre el mismo modelo de geometría de **chart-core** que
[Line chart](/es/components/line-chart/) (ADR-0004) — `@moderno/chart-core`,
construido sobre `d3-scale` + `d3-shape` — calculada una sola vez y renderizada como
SVG por cada paquete de framework. El color de la pista, el relleno y el texto viene
de tokens `--md-*`. Primitive **closed-prop** (ADR-0003) — se pasan `data` y props de
tamaño, sin anatomía composicional. `data` no se reordena, así que pásalo ya
clasificado para una lista de barras convencional.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { BarListChart } from '@moderno/react'

<BarListChart
  data={[
    { label: 'Directo', value: 456 },
    { label: 'Búsqueda orgánica', value: 351 },
    { label: 'Referencia', value: 271 },
  ]}
  width={480}
  height={220}
  label="Tráfico por fuente"
/>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `data` | `{ label: string; value: number }[]` | — | Las filas a graficar, en orden de presentación (no se reordenan). |
| `width` | `number` | `480` | Ancho intrínseco del SVG en px. El gráfico se escala al 100% de su contenedor vía CSS; `width`/`height` fijan la relación de aspecto (`viewBox`). |
| `height` | `number` | `240` | Alto intrínseco del SVG en px. |
| `label` | `string` | — | Etiqueta accesible leída por lectores de pantalla en lugar del gráfico visual. |
