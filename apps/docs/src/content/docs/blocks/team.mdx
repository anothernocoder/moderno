**Team** is a marketing [block](/introduction/blocks/): a team section with a centered
kicker/title/subtitle header above a responsive grid of member profile cards, each with an avatar,
name, role, and short bio. It composes the `Card` and `Avatar` primitives and the `--md-*` tokens.
You copy it into your repo with the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `kicker` | `string` | `'Nuestro equipo'` | Small text above the title |
| `title` | `string` | `'Conoce al equipo'` | Section title |
| `subtitle` | `string` | `'Las personas que diseñan y construyen Moderno todos los días.'` | Short intro text |
| `members` | `TeamMember[]` | Four sample members | The profile cards |

A `TeamMember` is `{ name: string; role?: string; bio?: string; avatarSrc?: string; initials?: string }`.
Each member renders as a `Card` whose body holds a centered `Avatar`, name, role, and bio. When
`avatarSrc` is absent, `Avatar` falls back to `initials`.

## Usage

```tsx
import { Team } from './blocks/Team'

<Team
  kicker="Team"
  title="Meet the team"
  subtitle="The people who design and build our product every day."
  members={[
    {
      name: 'Jamie Chen',
      role: 'Head of Design',
      bio: 'Leads the visual language and keeps it consistent across every surface.',
      initials: 'JC',
    },
    {
      name: 'Alex Rivera',
      role: 'Frontend Lead',
      bio: 'Owns the component architecture and API parity across frameworks.',
      initials: 'AR',
    },
  ]}
/>
```

The props API is identical across all four frameworks.

The block renders a centered header (kicker, title, subtitle) above a responsive grid of `Card`s
built from `members`, one card per person. The grid uses `repeat(auto-fit, minmax(240px, 1fr))`,
so cards reflow from multiple columns down to one as the viewport narrows.
