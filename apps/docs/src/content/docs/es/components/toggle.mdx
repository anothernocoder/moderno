**Toggle** es un interruptor on/off para ajustes binarios que se aplican al instante. Primitive **closed-prop** (ADR-0003) sobre la máquina `switch` de Zag.js: teclado, foco y ARIA vienen del behavior core; Moderno aporta el contrato de clases `md-*`.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Toggle } from '@moderno/react'

<Toggle label="Notificaciones" defaultChecked onCheckedChange={(c) => console.log(c)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Toggle } from '@moderno/vue'
import { ref } from 'vue'
const on = ref(true)
</script>

<template>
  <Toggle label="Notificaciones" v-model="on" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Toggle } from '@moderno/svelte'
  let on = $state(true)
</script>

<Toggle label="Notificaciones" bind:checked={on} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Toggle } from '@moderno/solid'

<Toggle label="Notificaciones" defaultChecked onCheckedChange={(c) => console.log(c)} />
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Texto visible junto al interruptor. |
| `defaultChecked` | `boolean` | `false` | Estado inicial no controlado. |
| `checked` / `v-model` / `bind:checked` | `boolean` | — | Estado controlado. |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback al cambiar (en Vue/Svelte usa el binding). |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `invalid` | `boolean` | `false` | Marca el control como inválido. |
| `name` / `value` | `string` | — | Para envío en formularios. |
