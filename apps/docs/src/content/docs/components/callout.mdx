## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Callout } from '@moderno/react'

<Callout variant="warning" title="Heads up">
  This action affects every member of the team.
</Callout>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Callout } from '@moderno/vue'
</script>

<template>
  <Callout variant="warning" title="Heads up">
    This action affects every member of the team.
  </Callout>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Callout } from '@moderno/svelte'
</script>

<Callout variant="warning" title="Heads up">
  This action affects every member of the team.
</Callout>
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Callout } from '@moderno/solid'

<Callout variant="warning" title="Heads up">
  This action affects every member of the team.
</Callout>
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Border color/intent. |
| `title` | `string` | — | Optional bold title. |
