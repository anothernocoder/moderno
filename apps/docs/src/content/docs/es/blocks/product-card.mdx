**Product Card** es un [block](/es/introduction/blocks/) de ecommerce: una tarjeta de producto
con imagen, badge de oferta opcional, título, precio y un CTA de "Agregar al carrito".
Compone los primitives `Card`, `Badge` y `Button` y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `image` | `string` | — | URL de la imagen (placeholder si se omite) |
| `title` | `string` | `'Silla Moderno'` | Nombre del producto |
| `price` | `string` | `'$320'` | Precio ya formateado |
| `saleText` | `string` | — | Texto del badge de oferta (oculto si se omite) |
| `onAdd` | `() => void` | — | Handler de "Agregar al carrito" |

## Uso

```tsx
import { ProductCard } from './blocks/ProductCard'

<ProductCard
  image="/img/silla.jpg"
  title="Silla Moderno"
  price="$256"
  saleText="-20%"
  onAdd={() => cart.add('silla')}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onAdd` es el evento
`@add`; en los demás es una prop callback.
