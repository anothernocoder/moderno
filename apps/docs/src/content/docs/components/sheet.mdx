**Sheet** is a slide-in drawer — a dialog variant (ADR-0003) for side or bottom panels
whose internal layout varies per consumer. It's **compositional**: parts share a single
Zag machine — `Sheet.Root`, `Sheet.Trigger`, `Sheet.Content`, `Sheet.Title`, `Sheet.Close`.
Behavior and a11y (focus trapping, `Esc` closes, click outside, ARIA) come from the same
`dialog` machine that powers [Dialog](/components/dialog/); Moderno only changes the
anatomy and adds the slide-in placement.

## Props

**`Sheet.Root`**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultOpen` | `boolean` | `false` | Initial open state (uncontrolled). |
| `open` / `v-model` / `bind:open` | `boolean` | — | Open state (controlled). |
| `onOpenChange` | `(open: boolean) => void` | — | Callback on open/close. |

**`Sheet.Content`**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `'right' \| 'left' \| 'top' \| 'bottom'` | `'right'` | Edge the sheet slides in from. |

`Sheet.Trigger` and `Sheet.Title` need no props beyond children. `Sheet.Close` also takes
an optional `closeLabel` (default `'Close'`) used as its `aria-label` when its children
are an icon rather than text.

## Accessibility

- Focus is trapped inside the sheet while it's open; on close, focus returns to the trigger.
- `Esc` and clicking the backdrop close the sheet.
- ARIA (`role="dialog"`, `aria-modal`, `aria-labelledby`) provided by Zag's `dialog` machine.
