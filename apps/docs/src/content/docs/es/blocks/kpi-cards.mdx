**KPI Cards** es un [block](/es/introduction/blocks/) de aplicación: tarjetas de métrica
primaria con valor, etiqueta e indicador de tendencia — la contraparte de una sola métrica
del bloque [Stats](/es/blocks/stats/). Compone los primitives `Card` y `Badge` y los
tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `KpiCardItem[]` | 3 KPIs de muestra | Métricas a renderizar |

### `KpiCardItem`

| Campo | Tipo | Default | Descripción |
|-------|------|---------|-------------|
| `label` | `string` | — | Nombre del KPI |
| `value` | `string` | — | Valor ya formateado |
| `delta` | `string` | — | Cambio vs. periodo anterior (oculto si se omite) |
| `trend` | `'up' \| 'down'` | — | Color y flecha del badge: `up` → success `↑`, `down` → error `↓` |
| `helpText` | `string` | — | Caption pequeño junto al delta (ej. "vs. mes anterior") |

## Uso

```tsx
import { KpiCards } from './blocks/KpiCards'

<KpiCards
  items={[
    { label: 'Ingresos recurrentes', value: '$128.4k', helpText: 'vs. mes anterior', delta: '+8.2%', trend: 'up' },
    { label: 'Tasa de cancelación', value: '2.3%', helpText: 'vs. mes anterior', delta: '-0.4%', trend: 'down' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.
