**Checkbox** is a boolean multi-select control that supports the **indeterminate** (mixed) state. It's a **closed-prop** primitive (ADR-0003) built on Zag.js's `checkbox` machine.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Checkbox } from '@moderno/react'

<Checkbox label="I accept the terms" onCheckedChange={(c) => console.log(c)} />
<Checkbox label="Mixed" defaultChecked="indeterminate" />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Checkbox } from '@moderno/vue'
import { ref } from 'vue'
const checked = ref(false)
</script>

<template>
  <Checkbox label="I accept the terms" v-model="checked" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Checkbox } from '@moderno/svelte'
  let checked = $state(false)
</script>

<Checkbox label="I accept the terms" bind:checked />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Checkbox } from '@moderno/solid'

<Checkbox label="I accept the terms" onCheckedChange={(c) => console.log(c)} />
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Text shown next to the box. |
| `defaultChecked` | `boolean \| 'indeterminate'` | `false` | Initial uncontrolled state. |
| `checked` / `v-model` / `bind:checked` | `boolean \| 'indeterminate'` | — | Controlled state. |
| `onCheckedChange` | `(checked) => void` | — | Callback fired on change. |
| `disabled` | `boolean` | `false` | Disables the control. |
| `invalid` | `boolean` | `false` | Marks the box as invalid. |
| `required` | `boolean` | `false` | Marks the field as required. |
| `name` / `value` | `string` | — | For form submission. |
