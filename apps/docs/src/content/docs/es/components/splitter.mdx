**Splitter** organiza paneles a lo largo de un eje con un disparador de redimensionado
arrastrable entre cada par de paneles adyacentes — paneles redimensionables, accesibles
por teclado. Primitive **closed-prop** (ADR-0003) sobre la máquina `splitter` de Zag.js:
se pasa un arreglo `panels` con ids y restricciones de tamaño, y se provee el contenido
de cada panel de la forma idiomática de cada framework.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Splitter } from '@moderno/react'

<Splitter
  panels={[
    { id: 'barraLateral', minSize: 20, maxSize: 50, children: <BarraLateral /> },
    { id: 'principal', children: <Principal /> },
  ]}
  defaultSize={[30, 70]}
/>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Splitter } from '@moderno/vue'
</script>

<template>
  <Splitter :panels="[{ id: 'barraLateral', minSize: 20, maxSize: 50 }, { id: 'principal' }]" :default-size="[30, 70]">
    <template #barraLateral><BarraLateral /></template>
    <template #principal><Principal /></template>
  </Splitter>
</template>
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `panels` | `SplitterPanelDef[]` | — | Los paneles a renderizar, en orden. Cada uno tiene un `id`, `minSize`/`maxSize`/`collapsedSize` opcionales (porcentajes), `collapsible` y `resizeBehavior`. React/Solid agregan `children` a cada panel; Vue/Svelte proveen el contenido mediante un slot/snippet nombrado según el `id` del panel. |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | El eje sobre el que se distribuyen los paneles. |
| `size` | `number[]` | — | Tamaños controlados de los paneles, como porcentajes. |
| `defaultSize` | `number[]` | — | Tamaños iniciales no controlados de los paneles, como porcentajes. |
| `onResize` | `(size: number[]) => void` | — | Se llama mientras se redimensiona un panel. Vue expone el mismo valor con `v-model:size` (`update:size`). |
| `onResizeStart` | `() => void` | — | Se llama al iniciar una interacción de redimensionado. Vue: evento `resizeStart`. |
| `onResizeEnd` | `(size: number[]) => void` | — | Se llama al terminar una interacción de redimensionado. Vue: evento `resizeEnd`. |
| `keyboardResizeBy` | `number` | — | Píxeles para redimensionar un panel al usar el teclado. |
| `resizeTriggerLabel` | `(beforeId: string, afterId: string) => string` | `` `Resize ${beforeId} and ${afterId}` `` | Etiqueta accesible para el disparador de redimensionado entre dos paneles. |
