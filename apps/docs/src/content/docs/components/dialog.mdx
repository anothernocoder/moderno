**Dialog** is a centered modal dialog. It's a **closed-prop** primitive (ADR-0003):
configured through props, not composed. The behavior — focus trapping, closing with `Esc`,
click outside, ARIA — comes from Zag's `dialog` machine; Moderno provides the markup and the classes.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `triggerLabel` | `string` | — | Text of the button that opens the dialog. |
| `title` | `ReactNode` / text | — | Dialog title. |
| `description` | `ReactNode` / text | — | Optional descriptive text. |
| `closeLabel` | `string` | `'Close'` | Text of the close button. |

## Accessibility

- Focus is trapped inside the dialog while it's open; on close, focus returns to the trigger.
- `Esc` and clicking the backdrop close the dialog.
- ARIA (`role="dialog"`, `aria-modal`, `aria-labelledby`, `aria-describedby`) provided by Zag.
