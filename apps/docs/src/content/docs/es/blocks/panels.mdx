**Paneles** es un [block](/es/introduction/blocks/) de aplicaciones: superficies de panel
genéricas para agrupar contenido dentro de una pantalla. Combina un encabezado (título +
descripción) con una grilla responsiva de superficies de panel con borde, cada una con un
título, un cuerpo y una acción de footer opcional. Compone los primitives `Card` y `Button` y
los tokens `--md-*` — sin primitives nuevos ni lógica de interacción a medida, solo un layout
copy-paste en el que puedes insertar contenido.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `title` | `string` | `'Ajustes de la cuenta'` | Encabezado del panel |
| `description` | `string` | Texto de ejemplo | Subtítulo del encabezado (se oculta si está vacío) |
| `panels` | `PanelItem[]` | 4 paneles de ejemplo | Las superficies de panel renderizadas en la grilla |
| `onPanelAction` | `(panel: PanelItem) => void` | — | Se invoca al hacer click en el botón de footer de un panel (React/Solid/Svelte); en Vue emite `panelAction` |

Forma de `PanelItem`:

| Campo | Tipo | Descripción |
|-------|------|--------------|
| `id` | `string` | Id único del panel / key de lista |
| `title` | `string` | Título del panel, renderizado con el slot `title` de `Card` |
| `body` | `string` | Texto del cuerpo del panel |
| `actionLabel` | `string?` | Etiqueta del botón de acción del footer del panel (se oculta si se omite) |

## Uso

```tsx
import { Panels } from './blocks/Panels'

<Panels
  title="Ajustes del espacio de trabajo"
  panels={[
    { id: 'profile', title: 'Perfil', body: 'Actualiza tu nombre, foto y biografía pública.', actionLabel: 'Editar perfil' },
    { id: 'security', title: 'Seguridad', body: 'Activa la verificación en dos pasos y revisa las sesiones activas.', actionLabel: 'Revisar' },
  ]}
  onPanelAction={(panel) => console.log('acción', panel.id)}
/>
```

La API de props es idéntica en los cuatro frameworks (Vue expone la acción del panel como un
emit `panelAction` en lugar de una prop `onPanelAction`, siguiendo las convenciones de eventos
de Vue). Cada panel es un `Card` dispuesto en una grilla
`grid-template-columns: repeat(auto-fit, minmax(220px, 1fr))`, por lo que los paneles se
reacomodan automáticamente al cambiar el ancho del contenedor. No hay arrastrar-para-reordenar,
colapsar/expandir ni otra lógica de interacción por encima; agrégala tú mismo una vez que lo
hayas copiado a tu repo.
