**Faq** es un [block](/es/introduction/blocks/) de marketing: una sección de preguntas frecuentes
con un encabezado centrado (kicker/título/subtítulo) sobre una lista de preguntas y respuestas.
Compone el primitive `Accordion` y los tokens `--md-*`. Lo copias a tu repo con el CLI y lo editas
libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'Preguntas frecuentes'` | Texto pequeño sobre el título |
| `title` | `string` | `'¿Tienes dudas?'` | Título de la sección |
| `subtitle` | `string` | `'Encuentra respuestas a las preguntas más frecuentes.'` | Texto de introducción corto |
| `items` | `FaqItem[]` | Cinco preguntas de ejemplo | La lista de preguntas y respuestas |
| `defaultValue` | `string[]` | `['faq-0']` | Ids de los items del accordion expandidos al montar |

Un `FaqItem` es `{ question: string; answer: string }`. Cada item se renderiza como un
`Accordion.Item` con un `value` generado como `faq-${index}`.

## Uso

```tsx
import { Faq } from './blocks/Faq'

<Faq
  kicker="Preguntas frecuentes"
  title="¿Tienes dudas?"
  subtitle="Respuestas a las preguntas que más recibimos."
  items={[
    { question: '¿Necesito tarjeta de crédito para empezar?', answer: 'No, el plan gratuito no requiere método de pago.' },
    { question: '¿Puedo cancelar en cualquier momento?', answer: 'Sí, desde la configuración de tu cuenta cuando quieras.' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.

El block renderiza un encabezado centrado (kicker, título, subtítulo) sobre un `Accordion.Root`
construido a partir de `items`, un `Accordion.Item` por pregunta. Por defecto el primer item está
expandido y el accordion es `collapsible`, así que cualquier item puede volver a cerrarse; pasa
`defaultValue={[]}` para empezar con todo colapsado.
