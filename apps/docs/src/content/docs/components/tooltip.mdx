**Tooltip** shows a contextual hint when you hover over or focus the trigger. A **closed-prop** primitive (ADR-0003) built on Zag.js's `tooltip` machine, positioned with popper. Like `Dialog`, it takes the trigger as `triggerLabel` (string) and renders its own button.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `triggerLabel` | `string` | — | Label of the trigger button. |
| `content` | `string` | — | The hint text. |
| `openDelay` | `number` | `400` | Open delay in ms. |
| `closeDelay` | `number` | `150` | Close delay in ms. |
