**Button** is the primary action control. It's a **closed-prop** and **pure CSS** primitive:
it uses no Zag machine, only the `md-*` class contract. One primary action per view.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary'` | `'primary'` | Visual hierarchy. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Control height (32 / 40 / 48 px). |
| `label` | `string` | — | Convenience text; if omitted, uses the children. |

It also accepts all native `<button>` attributes (`onClick`, `disabled`, `type`…).

## Accessibility

- Renders a native `<button>`: role, focus, and keyboard activation come for free.
- Visible focus uses a 2px outline in the `--md-primary` color.
- For icon-only buttons, always provide `aria-label`.
