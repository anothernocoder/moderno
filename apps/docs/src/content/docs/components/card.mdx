**Card** is a surface that groups related content and actions. A **closed-prop** primitive:
fixed structure (title, body, footer). In React/Solid the title and footer are props; in Vue
they are named slots (`#title`, `#footer`); in Svelte, snippets.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Card, Button } from '@moderno/react'

<Card
  title="Pro Plan"
  footer={<Button size="sm" label="Upgrade" />}
>
  Access to all features and priority support.
</Card>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Card, Button } from '@moderno/vue'
</script>

<template>
  <Card>
    <template #title>Pro Plan</template>
    Access to all features and priority support.
    <template #footer>
      <Button size="sm" label="Upgrade" />
    </template>
  </Card>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Card, Button } from '@moderno/svelte'
</script>

<Card>
  {#snippet title()}Pro Plan{/snippet}
  Access to all features and priority support.
  {#snippet footer()}
    <Button size="sm" label="Upgrade" />
  {/snippet}
</Card>
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Card, Button } from '@moderno/solid'

<Card
  title="Pro Plan"
  footer={<Button size="sm" label="Upgrade" />}
>
  Access to all features and priority support.
</Card>
```
</TabItem>

## Anatomy

| Part | React / Solid | Vue | Svelte |
|-------|---------------|-----|--------|
| Title | prop `title` | slot `#title` | snippet `title` |
| Body | children | default slot | children |
| Footer | prop `footer` | slot `#footer` | snippet `footer` |
