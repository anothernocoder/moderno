**Checkbox** es un control booleano de multi-selección que admite el estado **indeterminado** (mixto). Primitive **closed-prop** (ADR-0003) sobre la máquina `checkbox` de Zag.js.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Checkbox } from '@moderno/react'

<Checkbox label="Acepto los términos" onCheckedChange={(c) => console.log(c)} />
<Checkbox label="Mixto" defaultChecked="indeterminate" />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Checkbox } from '@moderno/vue'
import { ref } from 'vue'
const checked = ref(false)
</script>

<template>
  <Checkbox label="Acepto los términos" v-model="checked" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Checkbox } from '@moderno/svelte'
  let checked = $state(false)
</script>

<Checkbox label="Acepto los términos" bind:checked />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Checkbox } from '@moderno/solid'

<Checkbox label="Acepto los términos" onCheckedChange={(c) => console.log(c)} />
```

</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Texto visible junto a la casilla. |
| `defaultChecked` | `boolean \| 'indeterminate'` | `false` | Estado inicial no controlado. |
| `checked` / `v-model` / `bind:checked` | `boolean \| 'indeterminate'` | — | Estado controlado. |
| `onCheckedChange` | `(checked) => void` | — | Callback al cambiar. |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `invalid` | `boolean` | `false` | Marca la casilla como inválida. |
| `required` | `boolean` | `false` | Marca el campo como requerido. |
| `name` / `value` | `string` | — | Para envío en formularios. |
