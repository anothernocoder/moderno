**Portfolio Header** is a portfolio [block](/introduction/blocks/): an introductory header
with avatar, name, role, bio, social links, and a contact CTA. It composes the
`Avatar`, `Chip`, and `Button` primitives and the `--md-*` tokens.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | `'Ana Restrepo'` | Displayed name |
| `role` | `string` | `'Product designer'` | Role or title |
| `bio` | `string` | `'I build interfaces…'` | Short description |
| `avatarSrc` | `string` | — | Avatar URL (uses initials if omitted) |
| `initials` | `string` | `'AR'` | Avatar fallback |
| `links` | `PortfolioLink[]` | GitHub · Dribbble · LinkedIn | Social links |
| `onContact` | `() => void` | — | Contact CTA handler |

### `PortfolioLink`

| Field | Type | Description |
|-------|------|-------------|
| `label` | `string` | Link text |
| `href` | `string` | Destination URL |

## Usage

```tsx
import { PortfolioHeader } from './blocks/PortfolioHeader'

<PortfolioHeader
  name="Ana Restrepo"
  role="Product designer"
  initials="AR"
  links={[{ label: 'GitHub', href: 'https://github.com/ana' }]}
  onContact={() => openMailto()}
/>
```

The props API is identical across all four frameworks. In Vue, `onContact` is the
`@contact` event; in the others it's a callback prop.
