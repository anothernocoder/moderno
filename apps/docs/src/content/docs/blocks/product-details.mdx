**Product Details** is an ecommerce [block](/introduction/blocks/): a storefront product
detail page with an image gallery, rating, price with an optional compare-at price, a
stock badge, a variant selector, a quantity stepper, and an "Add to cart" CTA. It composes
the `Badge`, `Button`, and `NumberInput` primitives and the `--md-*` tokens. You copy it
into your repo with the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `images` | `string[]` | 3 placeholder slots | Gallery image URLs; the first is shown large, all are shown as thumbnails |
| `title` | `string` | `'Silla Roble Nórdica'` | Product name |
| `rating` | `number` | `4.8` | Star rating out of 5 |
| `reviewCount` | `number` | `132` | Number of reviews shown next to the rating |
| `price` | `string` | `'$389.000'` | Pre-formatted current price |
| `compareAtPrice` | `string` | `'$460.000'` | Pre-formatted original price, shown struck through (hidden if omitted) |
| `saleText` | `string` | `'-15%'` | Sale badge text (hidden if omitted) |
| `stockLabel` | `string` | `'En stock'` | Stock status label |
| `stockVariant` | `'neutral' \| 'solid' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'success'` | `Badge` color variant for the stock status |
| `description` | `string` | sample copy | Product description |
| `variantLabel` | `string` | `'Acabado'` | Label above the variant selector |
| `variants` | `string[]` | 3 sample finishes | Selectable variant options |
| `defaultVariant` | `string` | first of `variants` | Initially selected variant |
| `defaultQuantity` | `number` | `1` | Initial quantity |
| `onAddToCart` | `(payload: { variant?: string; quantity: number }) => void` | — | Called when "Add to cart" is clicked, with the selected variant and quantity |

## Usage

```tsx
import { ProductDetails } from './blocks/ProductDetails'

<ProductDetails
  images={['/img/chair-1.jpg', '/img/chair-2.jpg']}
  title="Oak Dining Chair"
  price="$389,000"
  compareAtPrice="$460,000"
  saleText="-15%"
  variants={['Light oak', 'Dark oak', 'Walnut']}
  onAddToCart={({ variant, quantity }) => cart.add({ variant, quantity })}
/>
```

The props API is identical across all four frameworks. In Vue, `onAddToCart` is the
`@add-to-cart` event; in the others it's a callback prop.
