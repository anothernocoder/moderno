**Hero** es un [block](/introduccion/blocks/) de marketing: una sección above-the-fold con
kicker, título, subtítulo y un CTA. Compone el primitive `Button` y los tokens `--md-*`.
Lo copias a tu repo con el CLI y lo editas libremente.

[importa los estilos](/introduccion/instalacion/). A partir de ahí el código es tuyo.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'Moderno'` | Texto pequeño sobre el título |
| `title` | `string` | `'Construye productos, no componentes.'` | Titular principal |
| `subtitle` | `string` | `'Un sistema de diseño…'` | Texto de apoyo |
| `ctaLabel` | `string` | `'Empezar ahora'` | Texto del botón CTA |
| `onCta` | `() => void` | — | Handler del click del CTA |

## Uso

```tsx
import { Hero } from './blocks/Hero'

<Hero
  kicker="Lanzamiento"
  title="Diseña más rápido"
  subtitle="Un look y un comportamiento para los cuatro frameworks."
  ctaLabel="Probar gratis"
  onCta={() => navigate('/signup')}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onCta` es el evento
`@cta`; en los demás es una prop callback.
