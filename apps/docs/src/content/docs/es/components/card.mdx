**Card** es una superficie que agrupa contenido y acciones relacionadas. Primitive
**closed-prop**: estructura fija (título, cuerpo, footer). En React/Solid el título y el
footer son props; en Vue son slots con nombre (`#title`, `#footer`); en Svelte, snippets.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Card, Button } from '@moderno/react'

<Card
  title="Plan Pro"
  footer={<Button size="sm" label="Actualizar" />}
>
  Acceso a todas las funciones y soporte prioritario.
</Card>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Card, Button } from '@moderno/vue'
</script>

<template>
  <Card>
    <template #title>Plan Pro</template>
    Acceso a todas las funciones y soporte prioritario.
    <template #footer>
      <Button size="sm" label="Actualizar" />
    </template>
  </Card>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Card, Button } from '@moderno/svelte'
</script>

<Card>
  {#snippet title()}Plan Pro{/snippet}
  Acceso a todas las funciones y soporte prioritario.
  {#snippet footer()}
    <Button size="sm" label="Actualizar" />
  {/snippet}
</Card>
```
</TabItem>
<TabItem label="Solid">

## Anatomía

| Parte | React / Solid | Vue | Svelte |
|-------|---------------|-----|--------|
| Título | prop `title` | slot `#title` | snippet `title` |
| Cuerpo | children | slot default | children |
| Footer | prop `footer` | slot `#footer` | snippet `footer` |
