**Action panels** es un [block](/es/introduction/blocks/) de aplicaciones: una lista de filas
estilo ajustes, cada una con título y descripción a la izquierda y una única acción a la
derecha — el clásico patrón "Danger Zone" para pantallas de cuenta/ajustes. Cualquier fila
puede activar un estilo destructivo (por ejemplo, para eliminar la cuenta). Compone los
primitives `Card`, `Button` y `Divider`, y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `items` | `ActionPanelItem[]` | 3 paneles de ejemplo | Las filas a renderizar |
| `onAction` | `(item: ActionPanelItem) => void` | — | Se llama con los datos de la fila al hacer clic en su botón de acción |

Forma de `ActionPanelItem`:

| Campo | Tipo | Descripción |
|-------|------|--------------|
| `title` | `string` | Título de la fila |
| `description` | `string` | Texto de apoyo bajo el título |
| `actionLabel` | `string` | Etiqueta del botón de acción único de la fila |
| `destructive` | `boolean` | Colorea el título y la acción con el token `--md-error` (estilo danger-zone) |

## Uso

```tsx
import { ActionPanels } from './blocks/ActionPanels'

<ActionPanels
  items={[
    { title: 'Descargar tus datos', description: 'Exporta todo en un archivo JSON.', actionLabel: 'Exportar' },
    {
      title: 'Eliminar cuenta',
      description: 'Esta acción es permanente y no se puede deshacer.',
      actionLabel: 'Eliminar cuenta',
      destructive: true,
    },
  ]}
  onAction={(item) => manejarAccionDelPanel(item)}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onAction` es el evento
`@action` en lugar de una prop callback.
