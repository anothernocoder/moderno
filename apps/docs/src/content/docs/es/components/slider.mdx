**Slider** ajusta un valor numérico dentro de un rango arrastrando el thumb o con el teclado. Primitive **closed-prop** (ADR-0003) sobre la máquina `slider` de Zag.js (un único thumb).

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Slider } from '@moderno/react'

<Slider label="Volumen" defaultValue={60} onValueChange={(v) => console.log(v)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Slider } from '@moderno/vue'
import { ref } from 'vue'
const volume = ref(60)
</script>

<template>
  <Slider label="Volumen" v-model="volume" />
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Slider } from '@moderno/svelte'
  let volume = $state(60)
</script>

<Slider label="Volumen" bind:value={volume} />
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Slider } from '@moderno/solid'

<Slider label="Volumen" defaultValue={60} onValueChange={(v) => console.log(v)} />
```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `label` | `string` | — | Etiqueta visible. |
| `defaultValue` | `number` | `50` | Valor inicial no controlado. |
| `value` / `v-model` / `bind:value` | `number` | — | Valor controlado. |
| `onValueChange` | `(value: number) => void` | — | Callback al cambiar. |
| `min` | `number` | `0` | Valor mínimo. |
| `max` | `number` | `100` | Valor máximo. |
| `step` | `number` | `1` | Incremento. |
| `disabled` | `boolean` | `false` | Desactiva el control. |
| `showValue` | `boolean` | `true` | Muestra el valor junto a la etiqueta. |
| `name` | `string` | — | Para envío en formularios. |
