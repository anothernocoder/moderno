**Area chart** plots `{x, y}` data as a filled SVG area with a stroked line along its
top edge, optionally with a dot per point. It builds on the same **chart-core**
geometry as [Line chart](/components/line-chart/) (ADR-0004) ‚Äî `@moderno/chart-core`,
built on `d3-scale` + `d3-shape` ‚Äî computed once and rendered as SVG by each framework
package. The fill closes to the bottom of the plot (not the data's y=0), so it reads
correctly even when every value is positive. Fill and stroke color come from
`--md-*` tokens. A **closed-prop** primitive (ADR-0003) ‚Äî pass `data` and size props,
no compositional anatomy.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { AreaChart } from '@moderno/react'

<AreaChart
  data={[
    { x: 0, y: 12 },
    { x: 1, y: 28 },
    { x: 2, y: 18 },
  ]}
  width={480}
  height={220}
  showPoints
  label="Monthly revenue"
/>
```
</TabItem>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `{ x: number; y: number }[]` | ‚Äî | The points to plot, in x order. |
| `width` | `number` | `480` | Intrinsic SVG width in px. The chart scales to 100% of its container via CSS; `width`/`height` set the aspect ratio (`viewBox`). |
| `height` | `number` | `240` | Intrinsic SVG height in px. |
| `showPoints` | `boolean` | `false` | Render a dot at each data point. |
| `label` | `string` | ‚Äî | Accessible label read by screen readers in place of the visual chart. |
