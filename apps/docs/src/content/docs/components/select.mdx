**Select** picks one option (or several) from a dropdown list. A **compositional** primitive
(ADR-0003): `Select.Root`, `Select.Label`, `Select.Trigger`, `Select.Content`, `Select.Item`.
The behavior (keyboard, typeahead, focus, ARIA `listbox`/`option`, popper positioning)
comes from Zag's `select` machine. `Root` takes `items` and builds the collection internally.

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Select } from '@moderno/react'

const fruits = [
  { label: 'Apple', value: 'apple' },
  { label: 'Banana', value: 'banana' },
]

<Select.Root items={fruits} onValueChange={(v) => console.log(v)}>
  <Select.Label>Fruit</Select.Label>
  <Select.Trigger placeholder="Pick a fruit" />
  <Select.Content>
    {fruits.map((f) => <Select.Item key={f.value} item={f} />)}
  </Select.Content>
</Select.Root>
```
</TabItem>
<TabItem label="Vue">

## Props (`Select.Root`)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `{ label: string; value: string; disabled?: boolean }[]` | — | Available options. |
| `defaultValue` | `string[]` | — | Initially selected keys (uncontrolled). |
| `value` / `v-model` / `bind:value` | `string[]` | — | Selected keys (controlled). |
| `multiple` | `boolean` | `false` | Allows multiple selection. |
| `disabled` | `boolean` | `false` | Disables the control. |
| `onValueChange` | `(value: string[]) => void` | — | Callback when the selection changes. |

`Select.Trigger` accepts `placeholder`. `Select.Item` takes `item` (an object from `items`).
