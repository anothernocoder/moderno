**Header** is a marketing [block](/introduction/blocks/): a site header / marketing
navigation bar with a logo, nav links, and CTA button(s). It composes the `Button`
primitive and the `--md-*` tokens. You copy it into your repo with the CLI and edit it
freely.

[import the styles](/introduction/installation/). From there, the code is yours.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `logoLabel` | `string` | `'Moderno'` | The logo text |
| `logoHref` | `string` | `'#'` | Where the logo links to |
| `navItems` | `{ label: string; href: string }[]` | Three sample links | The nav links shown in the top bar |
| `secondaryLabel` | `string` | `'Iniciar sesión'` | Label for the secondary text link. Pass an empty string to hide it |
| `secondaryHref` | `string` | `'#'` | Where the secondary link points to |
| `ctaLabel` | `string` | `'Empezar ahora'` | Label for the primary CTA. Pass an empty string to hide the CTA entirely |
| `ctaHref` | `string?` | — | Renders the CTA as a link instead of a button |
| `onCtaClick` | `() => void` | — | Called when the CTA button is clicked (ignored when `ctaHref` is set) |

## Usage

```tsx
import { Header } from './blocks/Header'

<Header
  logoLabel="Acme"
  navItems={[
    { label: 'Product', href: '/product' },
    { label: 'Pricing', href: '/pricing' },
    { label: 'Docs', href: '/docs' },
  ]}
  secondaryLabel="Sign in"
  secondaryHref="/sign-in"
  ctaLabel="Get started"
  ctaHref="/sign-up"
/>
```

The props API is identical across all four frameworks. In Vue, `onCtaClick` is the `@cta`
event; in the others it's a callback prop.

The bar itself wraps to a second line on narrow viewports (`flex-wrap`) rather than
collapsing into a hamburger menu — add a disclosure/menu primitive around the nav if you
need one.
