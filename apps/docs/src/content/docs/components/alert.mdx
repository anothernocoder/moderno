title: Alert
description: Inline feedback message (info/success/warning/error). Pure CSS, closed-prop.
---

import { Tabs, TabItem } from '@astrojs/starlight/components'
import { Alert as RAlert } from '@moderno/react'
import { Alert as VAlert } from '@moderno/vue'
import { Alert as SAlert } from '@moderno/svelte'
import { Alert as OAlert } from '@moderno/solid'

**Alert** renders inline feedback for the current view. A **closed-prop** primitive, pure
CSS — no Zag machine, consistent with [Callout](/components/callout/).

## Usage

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Alert } from '@moderno/react'

<Alert variant="error" title="Error">
  The change could not be saved: minimum stock cannot exceed the reorder point.
</Alert>
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Alert } from '@moderno/vue'
</script>

<template>
  <Alert variant="error" title="Error">
    The change could not be saved: minimum stock cannot exceed the reorder point.
  </Alert>
</template>
```
</TabItem>
<TabItem label="Svelte">
```svelte
<script lang="ts">
  import { Alert } from '@moderno/svelte'
</script>

<Alert variant="error" title="Error">
  The change could not be saved: minimum stock cannot exceed the reorder point.
</Alert>
```
</TabItem>
<TabItem label="Solid">
```tsx
import { Alert } from '@moderno/solid'

<Alert variant="error" title="Error">
  The change could not be saved: minimum stock cannot exceed the reorder point.
</Alert>
```
</TabItem>
</Tabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Intent, drives the border color. |
| `title` | `string` | — | Optional bold title. |
