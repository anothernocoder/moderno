**Marketing Stats** es un [block](/es/introduction/blocks/) de marketing: una sección de
métricas/números destacados con un encabezado centrado (kicker/título/subtítulo) sobre una fila de
números grandes con etiquetas. Es la variante de marketing de un block de estadísticas — distinta
del block [`Stats`](/es/blocks/stats/) de applications, que es un widget de dashboard KPI con
`Card`+`Badge`. Este no tiene cards ni badges: es una fila "de presunción" más grande y editorial,
pensada para una landing page, compuesta solo con los tokens `--md-*`. Lo copias a tu repo con el
CLI y lo editas libremente.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'En números'` | Texto pequeño sobre el título |
| `title` | `string` | `'El impacto de Moderno'` | Título de la sección |
| `subtitle` | `string` | `'Un solo design system, cuatro frameworks, resultados medibles.'` | Texto de introducción corto |
| `items` | `StatItem[]` | Cuatro estadísticas de ejemplo | Los items de métricas |

Un `StatItem` es `{ value: string; label: string }`. Cada item se renderiza como un número grande
en serif sobre una etiqueta más pequeña en mayúsculas, centrado, sin wrapper `Card` ni `Badge`.

## Uso

```tsx
import { MarketingStats } from './blocks/MarketingStats'

<MarketingStats
  kicker="En números"
  title="El impacto de Moderno"
  subtitle="Un solo design system, cuatro frameworks, resultados medibles."
  items={[
    { value: '4', label: 'Frameworks soportados' },
    { value: '30+', label: 'Componentes' },
    { value: '100%', label: 'Tokens compartidos' },
    { value: '0', label: 'Dependencias de UI extra' },
  ]}
/>
```

La API de props es idéntica en los cuatro frameworks.

El block renderiza un encabezado centrado (kicker, título, subtítulo) sobre una fila centrada y
envolvente de items de estadísticas construida a partir de `items`, con un ancho máximo de
`960px`. Los items usan `flex-wrap` con un gap de `48px`, así que se reacomodan en varias líneas a
medida que el viewport se angosta.
