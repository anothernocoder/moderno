**Hero** is a marketing [block](/introduction/blocks/): an above-the-fold section with a
kicker, title, subtitle, and a CTA. It composes the `Button` primitive and the `--md-*` tokens.
You copy it into your repo with the CLI and edit it freely.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `kicker` | `string` | `'Moderno'` | Small text above the title |
| `title` | `string` | `'Build products, not components.'` | Main headline |
| `subtitle` | `string` | `'A design system…'` | Supporting text |
| `ctaLabel` | `string` | `'Get started'` | CTA button text |
| `onCta` | `() => void` | — | CTA click handler |

## Usage

```tsx
import { Hero } from './blocks/Hero'

<Hero
  kicker="Launch"
  title="Design faster"
  subtitle="One look and one behavior across all four frameworks."
  ctaLabel="Try it free"
  onCta={() => navigate('/signup')}
/>
```

The props API is identical across all four frameworks. In Vue, `onCta` is the `@cta`
event; in the others it's a callback prop.
