**Radio** ofrece selección única dentro de un grupo de opciones mutuamente excluyentes. Primitive **closed-prop** (ADR-0003) sobre la máquina `radio-group` de Zag.js: pásale un array `options` y un `label` de grupo.

## Uso

<Tabs syncKey="framework">
<TabItem label="React">
```tsx
import { Radio } from '@moderno/react'

const options = [
  { value: 'card', label: 'Tarjeta' },
  { value: 'cash', label: 'Efectivo' },
]

<Radio label="Método de pago" options={options} defaultValue="card" onValueChange={(v) => console.log(v)} />
```
</TabItem>
<TabItem label="Vue">
```vue
<script setup lang="ts">
import { Radio } from '@moderno/vue'
import { ref } from 'vue'
const value = ref('card')
const options = [
  { value: 'card', label: 'Tarjeta' },
  { value: 'cash', label: 'Efectivo' },
]
</script>

<template>
  <Radio label="Método de pago" :options="options" v-model="value" />
</template>
```
</TabItem>
<TabItem label="Svelte">

```svelte
<script lang="ts">
  import { Radio } from '@moderno/svelte'

```
</TabItem>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `options` | `{ value: string; label?: string; disabled?: boolean }[]` | — | Opciones del grupo. |
| `label` | `string` | — | Etiqueta del grupo. |
| `defaultValue` | `string` | — | Valor inicial no controlado. |
| `value` / `v-model` / `bind:value` | `string` | — | Valor controlado. |
| `onValueChange` | `(value: string \| null) => void` | — | Callback al cambiar. |
| `disabled` | `boolean` | `false` | Desactiva todo el grupo. |
| `invalid` | `boolean` | `false` | Marca el grupo como inválido. |
| `orientation` | `'horizontal' \| 'vertical'` | `'vertical'` | Disposición de las opciones. |
| `name` | `string` | — | Para envío en formularios. |
