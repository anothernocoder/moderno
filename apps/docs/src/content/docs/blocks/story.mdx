**Story** es un [block](/introduccion/blocks/) de images-ads: una plantilla vertical 9:16
para historias sociales y anuncios, con kicker, título y CTA. Compone el primitive `Button`
y los tokens `--md-*`.

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `kicker` | `string` | `'Nueva colección'` | Texto pequeño superior |
| `title` | `string` | `'Diseño que se siente moderno.'` | Titular |
| `ctaLabel` | `string` | `'Comprar ahora'` | Texto del CTA |
| `onCta` | `() => void` | — | Handler del CTA |

## Uso

```tsx
import { Story } from './blocks/Story'

<Story
  kicker="Nueva colección"
  title="Diseño que se siente moderno."
  ctaLabel="Comprar ahora"
  onCta={() => navigate('/shop')}
/>
```

La API de props es idéntica en los cuatro frameworks. En Vue, `onCta` es el evento
`@cta`; en los demás es una prop callback.
